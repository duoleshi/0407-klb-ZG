# 重工施工方案AI智能审核系统(自研)

基于 AI 的工程方案（危大工程专项施工方案）智能审核系统，支持自动识别专业类型、智能匹配知识库规范、生成结构化审核报告。

## 文档导航

- [用户手册](./docs/用户手册.md) - 详细的功能介绍和操作指南
- [常见问题 FAQ](./docs/FAQ.md) - 使用过程中常见问题的解答

## 功能特性

- **智能专业识别** - 自动识别 13 个工程专业类型
  - 基坑工程、模板支撑体系工程、脚手架工程
  - 起重吊装及起重机械安装拆卸工程、拆除爆破工程、暗挖工程
  - 建筑幕墙安装工程、人工挖孔桩工程、钢结构安装工程
  - 水下作业工程、装配式建筑混凝土预制构件安装工程
  - 采用新技术新工艺新材料新设备工程、有限空间作业

- **知识库智能匹配** - 根据方案内容自动匹配相关规范文件
- **强制性标准优先加载** - 优先加载《智能体审核依据》强制性标准
- **多格式 PDF 解析** - 支持文本 PDF 和扫描版 PDF（OCR）
- **结构化审核报告** - 按标准格式输出审核意见
- **审核历史记录** - 保存历史审核记录，支持导出

## 技术栈

- **前端**: Next.js 16 + React 19 + TypeScript + Tailwind CSS
- **UI 组件**: Radix UI + shadcn/ui
- **AI 模型**: DeepSeek API
- **OCR 识别**: MinerU API + 百度 OCR + Tesseract.js（多级降级）
- **数据库**: SQLite (sql.js)
- **PDF 解析**: pdf-parse + pdfjs-dist

## 安装部署

### 环境要求

- Node.js 18+
- pnpm (推荐) 或 npm

### 安装步骤

1. **克隆仓库**
```bash
git clone https://github.com/duoleshi/Knowledge-Base-0317.git
cd Knowledge-Base-0317
```

2. **安装依赖**
```bash
pnpm install
```

3. **配置环境变量**

创建 `.env` 文件：
```env
# DeepSeek API Configuration
DEEPSEEK_API_KEY=your_deepseek_api_key
DEEPSEEK_BASE_URL=https://api.deepseek.com

# Baidu OCR Configuration (可选)
BAIDU_OCR_API_KEY=your_baidu_ocr_api_key
BAIDU_OCR_SECRET_KEY=your_baidu_ocr_secret_key

# MinerU API Configuration (可选，用于扫描版 PDF)
MINERU_API_TOKEN=your_mineru_api_token
```

4. **启动开发服务器**
```bash
pnpm dev
```

5. **访问应用**

打开浏览器访问 http://localhost:3000

## 使用说明

### 上传审核

1. 在首页点击上传区域，选择工程方案 PDF 文件
2. 点击"开始审核"按钮
3. 等待 AI 分析完成（约 30-90 秒）
4. 查看生成的审核报告

### 审核报告格式

审核报告按以下章节结构输出：

1. **审核依据** - 列出审核所依据的法律、法规、标准、规范
2. **通用性审核** - 按法律→法规→标准规范顺序输出
3. **严重缺陷审核** - 检查是否存在缺漏项
4. **内容完整性审核** - 检查方案内容完整性
5. **专业性审核** - 专业性技术问题审核

### 历史记录

- 点击导航栏"历史记录"查看所有审核记录
- 支持导出 TXT、MD、HTML 格式
- 支持删除记录

| 9 | 钢结构安装工程 | 钢结构、钢构件、钢梁、焊接 |
