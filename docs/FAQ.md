# 重工施工方案AI智能审核系统(自研) - 常见问题（FAQ）

本文档收集了用户在使用过程中经常遇到的问题及解答。如果您在使用过程中遇到问题，请先查阅本页面。

---

## 使用相关

### Q1：支持哪些文件格式？

系统支持以下文件格式：

| 格式 | 说明 |
|------|------|
| PDF | 支持文本型 PDF 和扫描版 PDF（自动 OCR 识别） |
| DOCX | Microsoft Word 文档（.docx 格式） |

> **提示**：暂不支持旧版 .doc 格式，请将 .doc 文件转换为 .docx 或 PDF 后上传。

---

### Q2：文件大小有限制吗？

建议单个文件大小不超过 **50MB**。

过大的文件可能导致：
- 上传时间较长
- 解析处理时间较长
- 部分浏览器可能出现卡顿

如果您的文件较大，建议：
- 压缩 PDF 文件大小
- 拆分为多个文件分别审核

---

### Q3：审核需要多长时间？

审核时间通常在 **30-90 秒**之间，具体取决于：

- 文件大小和页数
- 方案内容的复杂程度
- 当前服务器负载情况

审核过程中页面会显示处理状态，请耐心等待，**请勿关闭或刷新页面**。

---

### Q4：审核失败怎么办？

如果审核失败，请按以下步骤排查：

1. **检查文件格式**
   - 确认文件是 PDF 或 DOCX 格式
   - 确认文件未损坏，可以正常打开

2. **检查文件内容**
   - 确认文件内容非空
   - 如果是扫描版 PDF，请确保扫描清晰

3. **重试操作**
   - 删除已上传的文件
   - 重新上传并开始审核

4. **联系支持**
   - 如多次尝试仍失败，请联系技术支持
   - 请提供失败的文件名称和错误提示信息

---

### Q5：上传的文件会保存吗？

系统会在审核过程中临时存储文件内容用于分析，审核完成后：

- 文件内容不会被长期保存
- 系统仅保存审核记录（文件名、审核时间、审核结果）
- 您可以随时在历史记录中删除审核记录

---

## 审核相关

### Q6：审核报告怎么看？

审核报告按以下结构组织：

| 章节 | 内容说明 |
|------|----------|
| 审核依据 | 列出审核所依据的法律、法规、标准、规范 |
| 通用性审核 | 检查方案是否符合通用要求 |
| 严重缺陷审核 | 检查是否存在重要缺漏项 |
| 内容完整性审核 | 检查方案内容是否完整 |
| 专业性审核 | 针对专业技术的具体审核意见 |

建议**按章节顺序阅读**，优先关注「严重缺陷审核」和「专业性审核」中的问题。

---

### Q7：如何理解审核意见？

审核意见通常包含以下几类：

| 意见类型 | 说明 | 处理建议 |
|----------|------|----------|
| 符合要求 | 方案该部分满足规范要求 | 无需修改 |
| 建议改进 | 方案存在小问题，不影响安全 | 可选择性修改 |
| 需要修改 | 方案存在不符合规范的地方 | 需要修改后重新提交 |
| 缺漏项 | 方案缺少必要内容 | 需要补充完整 |

---

### Q8：专业类型选错了怎么办？

系统会**自动识别**方案所属的专业类型，无需手动选择。

如果自动识别结果与实际不符：
- 审核报告仍具有参考价值
- 建议修改文件名或方案内容，使其更明确地体现专业类型
- 重新上传进行审核

---

### Q9：为什么审核报告引用的规范文件不一样？

系统会根据方案内容**智能匹配**相关的技术规范和标准。不同专业类型、不同内容的方案，匹配到的规范文件会有所不同，这是正常现象。

## 其他

### Q10：是否支持批量审核？

目前系统**不支持批量审核**，需要逐个文件上传审核。

如有批量审核需求，欢迎向我们反馈，我们会根据需求量考虑后续开发。

---

### Q11：能否自定义审核标准？

目前系统使用内置的知识库进行审核，**暂不支持自定义审核标准**。

如果您有特定的审核需求，欢迎联系我们进行沟通。

---

我们非常欢迎您的反馈！

您可以通过以下方式联系我们：

### Q13：系统支持哪些浏览器？

系统支持主流的现代浏览器：

| 浏览器 | 支持版本 |
|--------|----------|
| Chrome | 最新版及最近两个大版本 |
| Firefox | 最新版及最近两个大版本 |
| Edge | 最新版及最近两个大版本 |
| Safari | 最新版及最近两个大版本 |

> **建议**：为获得最佳体验，请使用最新版本的 Chrome 浏览器。

---

### Q14：审核报告可以用于正式场合吗？

本系统生成的审核报告仅供**参考**，不能直接作为正式审核文件使用。

建议：
- 将系统审核报告作为初审参考
- 结合人工专业判断进行最终审核
- 正式审核请遵循相关法规和流程
