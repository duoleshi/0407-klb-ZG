# 前言

本文件按照GB/T1.1—2020《标准化工作导则第1部分：标准化文件的结构和起草规则》的规定起草。

本文件代替GB/T26557—2011《吊笼有垂直导向的入货两用施工升降机》。本文件与GB/T26557—2011相比，主要的技术性差异如下：

本文件的适用范围不再包括液压式和伸缩连杆式施工升降机；更改了规范性引用文件的年代号（见第2章，2011年版的第2章）；更改了“制动距离”术语，分为“制动系统制动距离”和“安全装置制动距离”两种情况进行了定义（见3.16和3.17，2011年版的3.18）；增加了对导轨架安装垂直度偏差的要求（见5.4.1.5）；将缓冲器试验时吊笼的运行速度由“额定速度加上  $0.2\mathrm{m / s}$  修改为“额定速度加上  $0.2\mathrm{m / s}$  和额定速度加上  $0.4\mathrm{m / s}$  两种情况（见5.4.3.2，2011年版的5.4.3.2）；增加了对只有专职操作者（司机）才能操作升降机时对全高度层门的规定（见5.5.5.1）；删除了“吊笼应配有有效的装置，以检测不稳固的导轨架节，并阻止吊笼驶入该导轨架节或保证吊笼不离开稳固的导轨架节”的要求（见2011年版的5.6.1.1）；将吊笼门开口的净高度“不应小于  $2.0\mathrm{m}$  修改为“不应小于  $1.8\mathrm{m}$  ”（见5.6.1.5.1.1，2011年版的5.6.1.4.1.1）；增加了“防止吊笼门脱离轨道”的要求（见5.6.1.5.3）；将“1.25倍的额定载荷的动载试验”修改为“1.1倍额定载荷的动载试验和1.25倍额定载荷的静载试验”（见6.3，2011年版的6.3）。

# 吊笼有垂直导向的人货两用施工升降机

# 1 范围

本文件规定了非曳引式人货两用施工升降机设计、制造、安装、使用、维护/检查的安全要求。

本文件适用于预定由允许进入建设施工工地的人员使用的、动力驱动的、临时安装的、带有服务于各层站的吊笼并具有下列特征的施工升降机（以下简称“升降机”）：

- 可运载人员或者人员和货物；- 吊笼有导向装置；- 吊笼垂直运行或沿着与垂直方向夹角最大不超过  $15^{\circ}$  的导轨运行；- 吊笼由卷筒驱动的钢丝绳悬挂或由齿轮齿条支持；- 导轨架架设时，需要或不需要其他独立结构物的支撑。

本文件不适用于：

曳引式施工升降机（见JB/T13031）；- 货用施工升降机（见GB/T10054.1、GB/T10054.2）；- 电梯（见GB/T7588.1、GB/T7588.2、GB21240、GB25194、GB/T25856和EN81- 43）；- 由起重机械悬挂的工作吊笼；- 叉车货叉承载的工作平台；- 移动式升降工作平台（见GB/T25849）和导架爬升式工作平台（见GB/T27547）；- 升降平台（见EN1570）；- 简易升降机（见GB28755）- 缆索车；- 专为军事用途设计的升降机；- 矿井升降机；- 舞台用升降设备；- 带液压驱动/制动系统和液压安全装置的升降机；- 特殊用途升降机；- 本文件实施日期之前生产的人货两用施工升降机。

本文件识别了此类设备在其寿命的各个阶段存在的危险，并描述了在按制造商的预定进行使用时消除或减小这些危险的方法（见第4章）。

本文件未规定对下列情况的附加要求：

- 恶劣状况（例如恶劣气候、强磁场）下的行动；- 防雷击；- 按特殊规则（如潜在爆炸性环境规则）采取的行动；- 电磁兼容性（发射、抗扰度）；- 可能导致危险情况的载荷（例如熔融的金属、酸/碱物质、辐射材料、易碎物品）的运送；- 内燃机的使用；- 遥控器的使用；- 制造过程中发生的危险；- 流动性（如转移场地）产生的危险；

架设在公共道路上引发的危险；

地震。

本文件涉及了升降机的安装。包括底架和底部防护围栏，但不包括任何混凝土、碎石、木材或其他基础设施的设计。包括附墙架的设计，但不包括锚固到支撑结构的锚定螺栓的设计。包括层门及其结构，但不包括锚固到支撑结构的任何锚定螺栓的设计。

# 3 术语和定义

GB/T15706界定的以及下列术语和定义适用于本文件。

3.1

# 施工升降机 buildershoist

带有有导向的平台、吊笼或其他运载装置，服务于建设施工工地各层站的临时安装的升降机械。

3.2

# 额定载荷 ratedload

额定载重量

升降机设计确定的工作状态下吊笼运载的最大载荷。

3.3

# 额定速度 rated speed

升降机设计确定的吊笼速度。

3.4

# 钢丝绳式升降机 wireropehoist

采用钢丝绳作为载荷悬挂系统的升降机。

3.5

# 强制式传动系统 positive drive

不通过摩擦力来传递驱动力的传动系统。

3.6

# 齿轮齿条式升降机 rack andpinionhoist

采用齿轮齿条作为载荷悬挂系统的升降机。

3.7

# 底架 base frame

用来支承和安装升降机其他所有组成部分的升降机最下部的结构件。

3.8

# 导轨 guides

确定吊笼或对重（有对重时）运行路线的刚性元件。

3.9

# 导轨架 mast

支撑和引导吊笼、对重（有对重时）的结构件。

3.10

# 导轨架节 mastsection

标准节

组成导轨架的不可再分割的结构件。

3.11

# 附墙架 masttie

附着架

连接导轨架和建筑物或其他固定结构物，为导轨架提供侧向支撑的结构件。

3.12

# 升降通道 hoistway

吊笼及其载荷运行的全部空间。

3.13

3.13对重通道 counterweight way对重运行的全部空间。

3.14

吊笼 cage

有底板、围壁、门和顶的运载装置。

3.15

对重 counterweight

平衡重

用于平衡其他重量的重物。

注：本文件中的“对重”，专指对吊笼起平衡作用的重物。

3.16

3.16制动系统制动距离 stopping distance of brake system; braking distance of brake system从控制或安全电路断开到吊笼完全停止运动，吊笼移动的距离。

3.17

3.17安全装置制动距离 stopping distance of safety device; braking distance of b safety device从防坠安全装置或超速安全装置开始动作到吊笼完全停止运动，吊笼移动的距离。

3.18

超速安全装置 overspeed safety device

使超速的吊笼或对重停止并保持停止状态的机械式装置。

3.19

松绳 slack rope

张紧的钢丝绳因外部载荷消除而处于的状态。

3.20

钢丝绳末端连接 wire rope termination

钢丝绳末端连接或固定所采用的方法和绳具。

3.21

层站 landing

建筑物或其他固定结构物上供人货出入吊笼的地点。

3.22

安全距离 safety distance

升降机上任一运动件与任一通道点之间允许的最小距离。

3.23

护栏 guard rail

除门之外，防止人员坠落或触及危险区域的固定式装置。

3.24

正常运行 normal operation

正常工作

升降机处于为运送载荷而使用的通常工作状态，但不包括维护/检查、安装、拆卸等工况。

3.25

工作状态 in service

升降机处于使用中的状态，无论吊笼在何位置、有无载荷、移动或静止。

3.26

非工作状态 out of service

已安装就位的升降机，其吊笼无载荷且处于受风影响最小的位置(通常但非一定处于地面)。

3.27

专业人员 competent person

为执行所需的工作程序而受过适当培训和必要指导，具有充分的知识和实践经验的指定人员。

# 5 安全要求和/或措施

# 5.1 通则

升降机的设计应考虑安全使用、安装、拆卸和维护/检查。升降机的安装应能使用安全的接近方法，例如通过吊笼顶或有同等效用的设施。

安装期间必须搬运的所有部件，例如导轨架节，其设计应考虑相应于人工搬运的重量。如果超出人工搬运的许用重量，则制造商应在使用说明书中推荐相关适用的起重设备。所有可移开和可拆卸的罩盖应采用不可脱离的紧固件固定。

施工升降机应符合本章规定的安全要求和/或防护措施。此外，对于本文件未涉及的相关但非重大危险，机器的设计应符合GB/T15706的原则。

# 5.2 载荷组合和计算

# 5.2.1 通则

升降机结构的设计和制造，应使其在所有预定工作工况下都有足够的强度，包括安装、拆卸和低温环境。

整机结构及其各部分的设计，应基于5.2规定的所有可能的载荷组合效应。载荷组合应考虑吊笼垂直运动和其任何水平移动期间，相对于导轨架及其附墙架最不利位置的吊笼和载荷。导轨架和支撑结构物之间的附墙架视为升降机结构的一部分。

# 5.2.2 载荷和力

# 5.2.2.1 不运动件的静载荷

除吊笼和随吊笼一起移动的设备之外，所有不运动件的静重。

# 5.2.2.2 运动件的静载荷

空载吊笼和所有随吊笼一起移动的设备的静重。

# 5.2.2.3 其他静载荷

由升降机支承的层站平台和层门的静重。

# 5.2.2.4 额定载荷

5.2.2.4.1 吊笼底板的额定载荷密度  $p$  按式(1)计算。

$$
p = F / A
$$

式中：

$p$  ——吊笼底板的额定载荷密度，单位为千牛每平方米  $(\mathrm{kN} / \mathrm{m}^{2})$ ； $F$  ——额定载荷，单位为千牛  $(\mathrm{kN})$ ； $A$  ——吊笼底板面积，单位为平方米  $(\mathrm{m}^{2})$ 。

额定载荷对吊笼和导轨架受力的影响，应按5.2.2.4.2和5.2.2.4.3的规定考虑。

5.2.2.4.2 如果  $p / 0.8 \leq 4.0 \mathrm{kN / m^2}$ ，则假定额定载荷分布在减小的面积  $(A_{1})$  上，其大小为额定载荷除以  $4.0 \mathrm{kN / m^2}$ ，其长、宽之比与吊笼底板的长、宽之比相同。该面积的形状和位置可视为使导轨和吊笼处于最不利的受力状态。见图1的示例。

装载和卸载时产生的力（见图4）应视为垂直方向和水平方向力的同时作用，每个方向的力计算如下：

吊笼门口；不是由层站支承的坡道或其他延伸面的前缘。同时，额定载荷的其余部分  $(F_{\mathrm{v1}},F_{\mathrm{v1}} = F - F_{\mathrm{v}})$  应作用于吊笼底板的中心。同样的力也应用于层站边缘和有关支撑结构的设计。使用说明书中应给出这些力的相关信息。

# 5.2.2.7 运行冲击系数

升降机运行时的运动载荷效应，为所有运动的载荷（吊笼、额定载荷、对重、钢丝绳等）的重量乘以冲击系数  $\mu = 1.1 + 0.264v$ ，其中  $v$  为额定速度，单位为  $\mathrm{m / s}$ 。如果其他系数被证实更准确，则可使用其他系数。

# 5.2.2.8 超速安全装置动作时的冲击系数

超速安全装置动作引起的力，为运动的载荷的总和乘以系数2.5。

如果在装载载荷最高到1.3倍额定载荷，包括传动系统的任何惯性效应的所有情况下进行的试验能验证该系数较小，则可采用这个较小的系数，但不应小于1.2。

# 5.2.2.9 站人吊笼顶承受的载荷

如果吊笼顶预定用于安装、拆卸、维护的通道或有紧急出口，则其设计应使其最不利的  $1.0\mathrm{m}^2$  面积上能承受至少  $3.0\mathrm{kN}$  的载荷。吊笼顶其他任一  $0.1\mathrm{m}\times 0.1\mathrm{m}$  区域还应能承受  $1.2\mathrm{kN}$  的载荷。

# 5.2.2.10 不站人吊笼顶承受的载荷

如吊笼顶不允许站人，则其任一  $0.1\mathrm{m}\times 0.1\mathrm{m}$  的区域应能承受  $1.0\mathrm{kN}$  的载荷。

# 5.2.2.11 吊笼底板承受的载荷

吊笼底板的设计应使其最不利的  $0.1\mathrm{m}\times 0.1\mathrm{m}$  区域能承受  $1.5\mathrm{kN}$  或  $25\%$  额定载荷（取两者中的较大值，但任何情况下均不大于  $3\mathrm{kN}$  的静力。

# 5.2.2.12 设计风载

# 5.2.2.12.1 通则

设计风载按GB/T3811或GB/T13752计算，但作用在吊笼上的风载计算应符合5.2.2.12.2的规定，风压按式（3）和5.2.2.12.3计算和确定。

任何情况下都假定风在任意的水平方向吹，计算时应考虑最不利方向。

$q = 0.625(v_{\mathrm{w}})^2$

式中：

$q$  ——风压，单位为牛每平方米  $(N / m^2)$ ，不同情况的风压，还应符合5.2.2.12.3； $v_{\mathrm{w}}$  ——风速，单位为米每秒  $(m / s)$ 。

# 5.2.2.12.2 作用在吊笼的风载

在计算作用在吊笼的风载时，应假定吊笼壁是实板，并应取其空气动力系数（风力系数）  $c = 1.2$  。系数1.2包括形状系数和挡风折减系数。

如果吊笼的设计允许将材料按5.6.1.4.3的规定伸出吊笼外，则应考虑附加的迎风面积，该面积应至少相当于一个从吊笼顶部开口开始向上延伸  $2\mathrm{m}$  高的实板箱体的迎风面积。

# 5.2.2.12.3 计算风压

# 5.2.2.12.3.1 通则

作用在升降机上的计算风压，应至少考虑  $5.2.2.12.3.2\sim 5.2.2.12.3.4$  所述的三种情况。

# 5.2.2.12.3.2 工作状态风压

工作状态风压不考虑高度，风压最小值应为  $q = 250\mathrm{N / m^2}$ ，对应风速  $v_{\mathrm{w}} = 20\mathrm{m / s}$ 。

# 5.2.2.12.3.3 非工作状态风压

非工作状态风压取决于升降机高于地面的高度和安装地区。

非工作状态风压及其高度变化系数推荐按GB/T3811—2008的表18和表19或GB/T13752—2017的表20和表21选取。

# 5.2.2.12.3.4 安装和拆卸工况风压

安装和拆卸工况风压不考虑高度，风压最小值应为  $q = 100 \mathrm{N / m^2}$ ，对应风速  $v_{\mathrm{w}} = 12.5 \mathrm{m / s}$ 。

# 5.2.2.13 安装垂直度误差

计算时应考虑至少  $0.5^{\circ}$  的安装垂直度误差。

# 5.2.2.14 对重的有利作用

在安装和拆卸过程中，不应考虑对重的有利作用。

# 5.2.2.15 缓冲器产生的作用力

应计算缓冲器产生的作用力。计算时，除非有经过验证的更低值，否则应采用  $1g$  的减速度。

# 5.2.6 传动系统和制动系统零部件的疲劳应力分析

5.2.6.1 应对易发生疲劳破坏的所有承载件和连接件进行疲劳应力分析，如轴和齿轮。分析时应考虑应力循环特性和应力循环次数，应力循环次数可以是载荷循环次数的数倍。

制造商应根据下列情况来确定应力循环次数：- 吊笼装有  $50\%$  的额定载荷进行80000次运动；- 吊笼空载进行80000次运动；- 应以每次运动（由静止到额定速度——以额定速度运行——减速至停止）的行程为  $20\mathrm{m}$  来进行传动系统的计算（见7.1.2.11）。

对于每一部件，应考虑向上和向下运动的最不利组合。

注：载人升降机运动的次数  $1.6 \times 10^{5}$  是按间歇式工作方式计算得出的（如10年，每年40周，每周40个小时，每小时运动10次）。

5.2.6.2 考虑到所有的应力集中效应，每根轴疲劳极限的最小安全系数应为2.0。

# 5.3 底架

5.3.1 底架应能承受升降机作用在其上的所有载荷，并能将载荷传递到其支承面上。

5.3.2 向支承面传递载荷时，不应通过任何弹性支承或充气轮胎。

5.3.3 使用可调节的方式将力传递到地面时，支脚应能在与水平面夹角至少为  $15^{\circ}$  的任何平面上自由转动，以防止结构中产生弯曲应力。如果支脚不能转动，则应考虑最不利的弯曲应力。

# 5.4 导轨架、附墙架和缓冲器

# 5.4.1 导轨和导轨架

5.4.1.1 导轨可以是导轨架的一部分。导轨应是刚性的，不应使用柔性元件，例如钢丝绳或链条。

应限制导轨架或吊笼任何构件的变形，以避免发生碰撞事故（例如与层站的碰撞）。

5.4.1.2 导轨或者导轨架应能承受5.2中的所有载荷情况。

5.4.1.3 导轨架或导轨之间的连接应能有效地传递载荷并保持对正。应只有在有意的手动操作下，才允许松动。

5.4.1.4 导轨/导轨架上的传动元件（如齿条）的紧固件，应能保证将传动元件保持在正确的位置，以能向导轨架传递规定的载荷，且应确保其固定不松动，例如使用锁紧螺母。

5.4.1.5 吊笼空载位于最低位置时，导轨架轴心线对底座水平基准面的安装垂直度偏差应符合表8的规定，但在设计允许倾斜的方向上除外。

# 5.4.2 附墙架

附墙架应能承受5.2规定的载荷情况。应特别注意安装和拆卸过程中产生的作用力。

# 5.4.3 缓冲器

5.4.3.1 应在吊笼和对重运行通道的最下方安装缓冲器。

5.4.3.2 缓冲器动作期间，吊笼的平均减速度不应大于  $1g$ ，且减速度峰值大于  $2.5g$  的时间不应超过  $0.04s$ （见5.2.2.15）。当吊笼作用在缓冲器上时的载荷情况和运行速度不超过下列规定时，都应满足前述要求：

a）吊笼装有额定载荷且运行速度等于额定速度加上  $0.2m / s$ ；或b）对于下列情况，吊笼装有额定载荷且运行速度等于额定速度加上  $0.4m / s$ ；- 非齿轮齿条传动系统；或- 带有单制动器的齿轮齿条传动系统；或- 驱动装置与吊笼分离（即驱动装置未安装在吊笼上）的升降机。

5.4.3.3 使用液压缓冲器时, 应提供检查油位的方法。应由电气安全开关监控液压缓冲器的动作, 当液压缓冲器被压缩时, 吊笼不能通过正常操作启动。

# 5.5 升降通道防护装置和层站入口

# 5.5.1 总则

为使用而安装的升降机, 应配有:底部防护围栏;各层站通道点处的层门;其他必要的升降通道(含对重通道)防护装置。

这些装置应防止人员被运动件撞击以及由层站/升降通道坠落。这些装置的设计要求见5.5, 正确使用说明见第7章, 验证见第6章。

# 5.5.2 升降机底部防护围栏

5.5.2.1 升降机底部防护围栏应围成一周, 高度不应小于  $2.0 \mathrm{~m}$ , 并应符合5.5.4和GB/T23821—2009中表1的要求。

底部防护围栏应设有围栏门。围栏门应视为全高度层门, 其开口的净高度不应小于  $2.0 \mathrm{~m}$ , 并应符合  $5.5.3.2 \sim 5.5.3.7, 5.5.3.8.1, 5.5.3.8.5, 5.5.3.8.6, 5.5.4, 5.5.5.1$  和5.5.5.3(其中的5.5.5.3.3除外)的要求。

5.5.2.2 所有吊笼和运动的对重都应在底部防护围栏的包围内。

5.5.2.3 维护时为能从底部防护围栏门出入, 围栏门应能从里面打开。

# 5.5.3 层站通道

# 5.5.3.1 通则

升降机安装时, 应在每个层站入口处, 包括底部防护围栏上, 安装层门。

# 5.5.3.2 层门的打开方向

层门不应朝升降通道打开。

# 5.5.3.3 实板层门

如果层门由实板材料制成, 则应能让使用者知道吊笼是否到达层站(见5.6.1.5.1.2)。

# 5.5.3.4 层门的导向装置

水平和垂直滑动门应有导向装置, 其运动应通过机械式限位装置限位。

# 5.5.3.5 层门的悬挂装置

垂直滑动的门应至少有两个独立的悬挂装置。柔性悬挂装置相应于其最小破断强度的安全系数不应小于6, 且应有将其保持在滑轮或链轮中的措施。

垂直滑动门用滑轮的直径不应小于其钢丝绳直径的15倍。钢丝绳绳端连接应符合5.7.3.2.1.5的规定。

层门用的平衡重应有导向装置, 且应能防止其滑出导轨, 即使在其悬挂失效的情况下。

门与平衡重的质量之差不应大于  $5 \mathrm{~kg}$  。

应有保护手指不被门板挤压的措施。

# 5.5.3.6 动力驱动的层门

5.5.3.6 动力驱动的层门动力驱动的层门，其动作和控制应符合GB/T7588.1—2020中5.3.6.2的规定，并应考虑雨、冰等环境的影响。

# 5.5.3.7 层门打开或关闭的操控

不应利用吊笼运动所操控的机械性装置或其他物件来打开或关闭层门。

5.5.3.8.1层门开口的净高度在层站上方不应小于  $2,0m$  。当建筑物入口的净高度小于  $2,0m$  时，允许降低层门开口的高度，但任何情况下层门在层站上方开口的净高度均不应小于  $1.8m$  0

5.5.3.9.1除了底部防护围栏外，可使用低高度层门。低高度层门应满足  $5.5.3.9.2\sim 5.5.3.9.8$  的要求。5.5.3.9.2层门高度应在  $1.1\mathrm{m}\sim 1.2\mathrm{m}$  。层门上部的内边缘与正常作业时的升降机任一运动件之间的安全距离（A，见图6和图7)不应小于  $0.85\mathrm{m}$  ；如果额定速度不大于  $0.7\mathrm{m / s}$  ，则此安全距离不应小于 $0.50\mathrm{m}$  。层门上部的外边缘与正常作业时的升降机运动件的安全距离（B，见图6和图7)不应小于 $0.75\mathrm{m}$  ；如果额定速度不大于  $0.7\mathrm{m / s}$  ，则此安全距离不应小于  $0.40\mathrm{m}$  。

5.5.3.9.3层门应全宽度遮住开口，并应至少由顶部栏杆、中间高度的横杆和高出地板至少  $150~\mathrm{mm}$  的护脚板组成，护脚板离地间隙不应大于  $35~\mathrm{mm}$  。如果在高度为  $1.1\mathrm{m}\sim 1.2\mathrm{m}$  的顶部栏杆下方，层门内边缘的任一部分与升降机运动件的距离小于  $0.5\mathrm{m}$  ，则层门上的任一开口均应不能穿过直径为  $50~\mathrm{mm}$  的球体。

5.5.3.9.4关闭的层门的外边缘与面向升降机的层站边缘之间的距离不应大于  $200~\mathrm{mm}$  。

5.5.3.9.5应设有层站通道侧面防护装置，该装置的高度应在  $1.1\mathrm{m}\sim 1.2\mathrm{m}$  ，并应有中间高度的横杆和至少高于地面  $150~\mathrm{mm}$  的护脚板，护脚板离地间隙不应大于  $35~\mathrm{mm}$  。

5.5.3.9.6应有自动减小吊笼边缘与层站边缘之间任何水平距离以及吊笼与层站通道侧面防护装置之间任何开口间距的措施，以使在层门打开之前以及吊笼在层站而层门处于打开状态期间，前述距离和间距不大于  $150~\mathrm{mm}$  。

5.5.3.9.7如果侧面防护装置是层站的一部分，并且在吊笼做垂直运动时与层门保持不小于  $0.85\mathrm{m}$  （额定速度大于  $0.7\mathrm{m / s}$  时)或不小于  $0.5\mathrm{m}$  （额定速度不大于  $0.7\mathrm{m / s}$  时)的安全距离，则吊笼和侧面防护装置之间的开口间距不应小于  $100~\mathrm{mm}$  。

5.5.3.9.8在装载和卸载时，应有措施使吊笼边缘和层站边缘之间的任何水平距离都不大于  $50~\mathrm{mm}$  。

# 5.5.4 升降通道防护装置的材料和净距

5.5.4.1 对于全高度层门，在其锁定的位置，用一个刚性的  $5000\mathrm{mm}^2$  的方形或圆形平坦表面将  $300N$  的法向力施加到门的任一面上的任何位置时，门应：

能承受且无永久变形；弹性变形不大于  $30~\mathrm{mm}$  试验之后工作正常。

当用一个刚性的  $5000\mathrm{mm}^2$  的方形或圆形平坦表面，将  $600N$  的法向力施加到门的任一面的任何位置时，门可不满足上述要求，但应保持安全。

5.5.4.2 对于低高度层门，当用  $1\mathrm{kN}$  的垂直力作用到层门顶部的任一位置，用  $300N$  的水平力作用在顶杆、中间杆、护脚板上任一位置时，门应：

能承受且无永久变形；试验之后工作正常。

5.5.4.3 其他升降通道防护装置应按其与全高度层门或低高度层门的相似情况，符合5.5.4.1或5.5.4.2的要求。

5.5.4.4 升降通道防护装置和层门关闭时，除其下部间隙不应大于  $35~\mathrm{mm}$  外，其与相邻运动件的间距有关的任何通孔和开口的尺寸及门周围的任何间隙，应符合GB/T23821—2009中表4的要求；但如果升降通道防护装置和层门与正常运行的升降机任一运动件之间的距离，在额定速度大于  $0.7\mathrm{m / s}$  时不小于  $0.85\mathrm{m}$  或者在额定速度不大于  $0.7\mathrm{m / s}$  时不小于  $0.5\mathrm{m}$ ，则升降通道防护装置和门应符合5.5.3.9中对低高度层门的有关要求。

# 5.5.5 层门门锁装置

# 5.5.5.1 全高度层门（见5.5.3.8）

5.5.5.1.1 如果被准许进入工地的人员都可操作升降机，则在正常运行工况下：

应不可能打开任何层门，除非吊笼底板距预定层站在  $\pm 0.15\mathrm{m}$  的垂直距离范围内；应不可能启动或保持吊笼运行，除非所有层门都处于关闭位置。载有额定载荷以额定速度运行的吊笼，如果其制动系统最大制动距离大于  $0.25\mathrm{m}$ ，则：应不可能打开任何层门，除非吊笼停止在距预定层站  $\pm 0.25\mathrm{m}$  的垂直距离范围内；且正常运行工况下应不可能启动或保持吊笼运行，除非所有层门都处于关闭和锁紧位置。

5.5.5.1.2 如果只有专职操作者(司机)才能操作升降机,则在正常运行工况下(见7.1.2.8和7.2.4):  ——应只有所有层门都处于关闭和锁紧位置时,吊笼才能启动或保持运行;  ——层门应只能由吊笼内的专职操作者(司机)来打开和锁闭,且层门的结构应保证,除5.5.5.1.3的紧急解锁外,在层站内侧无法打开层门。

5.5.5.1.3 紧急开锁:每一层门都应能用符合GB/T7588.1—2020中5.3.9.3规定的开锁钥匙从层站内侧解锁。

# 5.5.5.2 低高度层门(见5.5.3.9)

层门应配备可核验其关闭和锁紧位置的联锁装置。该联锁装置的动作应在吊笼门口处控制。用简单的方法应不能干扰该联锁装置的动作。

在正常运行工况下,除非所有的层门都已关闭和锁紧,否则应不可能启动或保持吊笼运行。

# 5.5.5.3 设计

5.5.5.3.1 门锁装置中的电气触点应为安全触点,见5.8.6。

5.5.5.3.2 所有全高度层门(见5.5.3.8)配备的门锁装置,以及所有相关的致动装置和电气触点,其安装位置或防护应只能使专业人员在层站上易于接近。

5.5.5.3.3 低高度层门(见5.5.3.9)配备的门锁装置,应只有借助工具才能使其电气安全装置不起作用。

5.5.5.3.4 所有门锁装置应安装牢靠,紧固件应有防松措施。

5.5.5.3.5 所有门锁装置和紧固件在其锁紧位置应能承受  $1\mathrm{kN}$  沿开门方向的力。

5.5.5.3.6 门锁装置的设计应使其可维修。不耐受粉尘或水的机械零部件,其防护等级不应低于GB/T4208中规定的IP44。

5.5.5.3.7 可拆式罩盖的拆除不应干涉任何锁紧机构或配线。所有可拆式罩盖应采用不可脱离的紧固件固定。

5.5.5.3.8 锁紧元件应通过弹簧或重力保持在锁紧位置。如果使用弹簧,则应是压缩弹簧且有导向。弹簧失效不应导致锁紧不安全。

5.5.5.3.9 应只有所有锁紧元件的接合长度不小于  $7\mathrm{mm}$  时,吊笼才能保持运行。

5.5.5.3.10 当打开全高度层门(见5.5.3.8)产生的间隙超过5.5.3.8.6的规定时,门锁装置中的电气触点应能阻止吊笼运行。

5.5.5.3.11 对于悬板式门锁装置,层门关闭后,悬板应与门扇全宽度重叠,足以在进行制造商预定的维护时防止层门打开。

# 5.5.6 净距

# 5.5.6.1 通则

本文件未规定的安全距离,应符合GB/T23821的规定。所有间隙应符合GB/T12265的规定。

# 5.5.6.2 吊笼下方的净空

在吊笼下方设置维护用安全通道时,应采取措施使其下方净空的垂直高度不小于  $1.8\mathrm{m}$  (例如通过可移动支架或等效工具)。该空间高度应延伸到吊笼的整个底面。该措施的安装和拆卸,不应需要任何人在吊笼下面进行。

# 5.6 吊笼

# 5.6.1 通则

# 5.6.1.1 基本要求

吊笼应完全封围。

在规定吊笼最大载人数时，应按人均占用吊笼底板面积  $0.2\mathrm{m}^2$  计算，人均体重应按  $75\mathrm{kg}$  计。

吊笼结构应根据5.2计算。

吊笼应有防止其脱离或卡住的刚性导向装置。

吊笼应有在其常规导靴或滚轮失效时仍使其保持在导轨上的有效装置。

应有防止吊笼驶出轨道的机械措施。这些措施在升降机正常运行、安装、拆卸或维护/检查时，均应起作用。

# 5.6.1.2 吊笼底板

吊笼底板应能承受5.2.2.11规定的力，并应能防滑（如采用花纹钢板）和自排水。

# 5.6.1.3 吊笼围壁

吊笼底板与吊笼顶之间应全高度有围壁，并应符合5.5.4.1的规定。围壁上的开口应符合GB/T23821—2009中表4的规定，且应不能穿过直径为  $25\mathrm{mm}$  的球体。任何危险的凸出物，均应按GB2894进行标志。

# 5.6.1.4 吊笼顶

5.6.1.4.1 吊笼应封顶。

5.6.1.4.2 吊笼内的净高度不应小于  $2.0\mathrm{m}$

5.6.1.4.3 运送较长材料时，如果能保证材料不伸到升降通道外，吊笼顶可开最大面积为  $0.15\mathrm{m}^2$  的开口。开口应设有盖门。

注：吊笼顶用作紧急出口的活板门不能用于运输长物料。

5.6.1.4.4 如果吊笼顶用于升降机自身的安装、拆卸、维护/检查或设有紧急出口，则顶板应防滑且周围应设护栏。

护栏应由上部栏杆、中间高度的横杆和护脚板等组成，上部栏杆应至少高出吊笼顶  $1.1\mathrm{m}$ ，护脚板高度不应小于  $150\mathrm{mm}$ 。在吊笼顶护栏封围的区域内，应可安全地进行安装、维护或检查作业。吊笼顶板边缘与护栏的水平距离不应大于  $200\mathrm{mm}$ 。

5.6.1.4.5 如果另一吊笼或对重的运动件与护栏内边缘的距离在  $0.3\mathrm{m}$  以内，则应对该运动件设置高度不少于  $2.0\mathrm{m}$  的附加护栏，且其每侧应比运动件宽出  $0.1\mathrm{m}$ 。

5.6.1.4.6 吊笼顶结构应根据5.2.2.9和5.2.2.10计算。

5.6.1.4.7 若笼顶有通孔，则应不能穿过直径为  $25\mathrm{mm}$  的球体。

# 5.6.1.5 吊笼门

# 5.6.1.5.1 手动门

5.6.1.5.1.1 吊笼门开口的净高度不应小于  $1.8\mathrm{m}$ ，净宽度不应小于  $0.6\mathrm{m}$ 。

门应能完全遮蔽开口。

门关闭时，除门下部间隙不应大于  $35\mathrm{mm}$  外，门上的通孔及门周围的间隙或零件间的间隙，应符合

GB/T23821—2009表4的要求，且应不能穿过直径为  $25\mathrm{mm}$  的球体。

5.6.1.5.1.2 实板门应有视窗，视窗面积不应小于  $25000\mathrm{mm}^2$  ，其尺寸和位置应让人能看见层站边缘。

5.6.1.5.1.3 门的设计应符合  $5.5.3.3\sim 5.5.3.7$  的要求。

5.6.1.5.1.4 所有的门都应配备机械锁，以使门在运行状态下不能打开；但对于被准许进入工地的人员都可操作的升降机，在其吊笼底板与预定层站的距离符合5.5.5.1.1要求时除外。

5.6.1.5.1.5 除非所有吊笼门都处于关闭位置，否则应不可能启动和保持吊笼运行。

5.6.1.5.1.6 吊笼门应能承受正常施加在其任何位置的  $300N$  推力，且不出现永久变形、不脱离其导向装置，弹性变形不应大于  $30\mathrm{mm}$  。  $300N$  的推力应通过面积为  $5000\mathrm{mm}^2$  的刚性方形或圆形平坦表面来施加。

5.6.1.5.1.7 在打开吊笼门之前，应采取措施满足5.5.3.8.3、5.5.3.9.6的要求，除非这些要求是通过打开吊笼门来实现的。

5.6.1.5.1.8 吊笼入口处所有机械和电气安全装置，其设计应符合5.5.5.3.1以及  $5.5.5.3.4\sim 5.5.5.3.11$  的要求。

5.6.1.5.1.9 吊笼门锁装置及其任何相关的致动装置和电气触点，其安装位置或防护，在所有吊笼门关闭的情况下，应使未经授权的人员难以从吊笼内接触到。

# 5.6.1.5.2 动力驱动的门

吊笼门如是动力驱动，则其动力操作系统应符合GB/T7588.1—2020中5.3.6.2的规定，并应考虑环境如雨、冰等的影响。

# 5.6.1.5.3 防止吊笼门脱离轨道

应有防止吊笼门因其导向装置失效而脱离轨道的措施。

# 5.6.1.6 紧急出口

5.6.1.6.1 对吊笼内乘员的救助总是应来自外部。5.10中提供了有关的紧急操作。

5.6.1.6.2 吊笼上应至少有一扇门或活板门用做紧急出口。这些门应可在吊笼外不借助钥匙打开，或在吊笼内用专用钥匙打开。紧急出口可以是吊笼门、吊笼顶活板门或其他紧急逃离门。

注：吊笼顶用作紧急出口的活板门不能用于运输长物料。

5.6.1.6.3 任何紧急出口门的锁紧，都应通过符合5.8.6的电气安全装置来验证。如果门未锁紧，则该装置应使升降机停止运行。应只有在重新锁紧后，才能恢复升降机的运行。

5.6.1.6.4 吊笼顶任何活板门的关闭，都应通过符合5.8.6的电气安全装置来验证。如果活板门未关闭，则该装置应使升降机停止运行。

5.6.1.6.5 设在围壁上的任何紧急出口门，其尺寸应至少为  $0.4\mathrm{m}\times 1.4\mathrm{m}$ ，并应向吊笼内打开或是滑动式的，或用其他方式提供通往导轨架或建筑结构物的安全通道。

5.6.1.6.6 设在吊笼顶的任何活板门，其尺寸应至少为  $0.4\mathrm{m}\times 0.6\mathrm{m}$ ，且不应向吊笼内打开。通往此类活板门的梯子应始终在吊笼内且可用。

# 5.6.2 吊笼防坠安全装置

5.6.2.1 应设有在吊笼超速时动作的防止吊笼坠落的超速安全装置。

5.6.2.2 安全装置在任何时候都应起作用，包括安装、拆卸和动作后重新设置之前。除齿条外，其他常规的传动件不应用于超速安全装置。

5.6.2.3 安全装置应能使装有1.3倍额定载荷的吊笼停止并保持停止状态。安全装置应根据5.2特别是5.2.2.8进行计算。

5.6.2.9 限速器用滑轮不应安装在悬挂吊笼的钢丝绳滑轮支承轴上。

5.6.2.10 超速安全装置不应借助于任何电气或气动装置来动作。

5.6.2.11 除超载外，在所有载荷情况下，安全装置动作后，吊笼底板相对于其正常位置的倾斜度不应大于  $5\%$  ，且应能恢复原状而无永久变形。

注：倾斜度是指吊笼底板倾斜角度的正切函数（tan）值。

5.6.2.12 安全装置的动作速度不应大于升降机额定速度加上  $0.4\mathrm{m / s}$  。

5.6.2.13 应有措施防止安全装置因外部物质的积聚或大气状况的影响而失效。

5.6.2.14 限速器用钢丝绳及其绳端连接等的尺寸和设计应符合5.7.3.2.1的要求。

在升降机安装期间，限速器用钢丝绳应直接悬挂在导轨架上。限速器动作时，限速器钢丝绳的张力应至少为下列的较大值：——300N，或——安全装置起作用所需力的两倍。

5.6.2.15 夹紧一个以上导轨的安全装置，应在所有导轨上同时起作用。

5.6.2.16 由弹簧来施加制动力的安全装置，其任一弹簧失效都不应导致安全装置危险失灵。

# 5.6.3 超载检测装置

5.6.3.1 应设有超载检测装置。在吊笼内载荷大于  $110\%$  额定载荷时，超载检测装置在吊笼内应给出清晰的警示信号，并应阻止其正常启动。

本文件规定了超载检测方法，但未要求配备载荷- 力矩检测装置，因为力矩可通过结合超载检测装置用稳定性和应力计算（见5.2）来确定。

不应为使用者设置取消警示信号的装置。

超载检测应至少在吊笼静止时进行。

5.6.3.2 超载指示器、检测器的设计和安装，应考虑在不拆卸和不影响指示器和检测器性能的情况下，满足升降机超载试验的需要。

5.6.3.3 如果动力中断，超载检测装置的所有数据和指示刻度应能保留。

5.6.3.4 应对超载检测装置加以保护，以防止其因冲击、振动、正常使用（包括安装、拆卸、维护/检查）和制造商预定的环境影响而损坏。

# 5.7 传动系统

# 5.7.1 通则

5.7.1.1 每一吊笼都应至少设有一套独立的传动系统。5.7.1.2 每一传动系统都应根据5.2包括5.2.6给出的具体要求计算。5.7.1.3 驱动电动机应通过不能脱开的强制式传动系统与卷筒或驱动齿轮连接。5.7.1.4 在正常运行时，吊笼应在动力作用下随时升降。5.7.1.5 在正常运行工况下，吊笼空载上升或额定载荷下降的速度不应大于其额定速度的  $115\%$

# 5.7.2 防护装置和可接近性

5.7.2.1 在正常运行期间，如果传动机构和相关装置的零部件的安全距离小于  $0.50\mathrm{m}$ ，则应按GB/T8196的规定对其进行防护。安全距离应符合GB/T23821和GB/T12265的规定。

5.7.2.2 应设置固定式防护装置，以防止进入可能损坏传动件的物质，如砂砾、雨、雪、冰、泥土或粉尘。

5.7.2.3 齿轮、皮带、链条、旋转轴、飞轮、滚轮、联轴器及类似的旋转件应有有效的防护装置。但通过设计或布置已使其安全，且被设计成在日常检查和维护时易接近的零部件除外。

防护装置关闭时，其与相邻运动件的间距有关的任何通孔和开口的尺寸，应符合GB/T23821—2009表4的要求。

# 5.7.3 悬挂系统

# 5.7.3.1 齿轮和齿条传动

# 5.7.3.1.1 通则

5.7.3.1.1.1 驱动齿轮和超速安全装置齿轮应直接固定在各自的轴上，不应采用摩擦和夹紧的方法连接。

5.7.3.1.1.2 安装时，安全装置齿轮应位于驱动齿轮之下。

5.7.3.1.1.3 齿条应可靠固定。齿条的接合处应对正，以避免错误啮合或损坏齿。

5.7.3.1.1.4 应有措施防止异物进入每一驱动齿轮或安全装置齿轮与齿条的啮合区间。

5.7.3.1.1.5 对其他齿类传动，如销齿条，同样应符合  $5.7.3.1.1\sim 5.7.3.1.4$  的规定，并应具有同样的安全系数。

# 5.7.3.1.2 设计

# 5.7.3.1.2.1 齿轮

齿轮设计应按GB/T3480.1、GB/T3480.2、GB/T3480.3和GB/T3480.5的规定考虑轮齿的弯曲强度和接触强度，并应考虑5.2.6的要求。

考虑了制造商使用说明书中规定的最大磨损量的齿轮弯曲疲劳强度安全系数不应小于2.0。

齿轮接触疲劳强度安全系数不应小于1.4。

# 5.7.3.1.2.2 齿条

齿条应采用与其啮合齿轮磨损情况相适应的材料制造，其设计应按GB/T3480.1、GB/T3480.2、GB/T3480.3和GB/T3480.5的规定考虑齿的弯曲强度和接触强度，并应考虑5.2.6的要求。

考虑了制造商使用说明书规定的最大磨损量的齿条静强度安全系数不应小于2.0。

# 5.7.3.1.2.3 载荷分配

当有多个驱动齿轮与齿条啮合时，应有自我调节措施有效地将载荷分配到每一驱动齿轮上，或者传动系统应设计成能适应所有正常工况下载荷在齿轮之间的分配。

# 5.7.3.1.3 模数

齿轮与齿条的模数应：

当传动系统中的背轮或其他啮合控制功能直接作用到齿条上而没有任何其他的导轨架组成件干预时，不小于4；当背轮或其他啮合控制装置通过与齿条直接接触的其他导轨架组成件而作用到齿条上时，不小于6。

# 5.7.3.2 悬挂吊笼和对重的钢丝绳

# 5.7.3.2.1 钢丝绳要求

5.7.3.2.1.1 悬挂用钢丝绳不应少于两根，且相互独立。若采用复绕法，应考虑钢丝绳的根数而不是其下垂的根分支数。

应设置自动平衡悬挂钢丝绳张力的装置。任何弹簧都应在压缩的状态下工作。

当单根钢丝绳过分拉长或破坏时，电气安全装置应停止升降机的运行（见5.9.3）。

5.7.3.2.1.2 钢丝绳直径不应小于  $8\mathrm{mm}$

5.7.3.2.1.3 钢丝绳的特性至少应符合GB/T20118的要求。

5.7.3.2.1.4 悬挂钢丝绳的安全系数应：

卷筒驱动的：  $\geq 12$  量悬挂对重的：  $\geq 6$  0

钢丝绳的安全系数为其最小破断载荷与最大静力之比。

5.7.3.2.1.5 钢丝绳绳端连接(固定)的强度不应小于钢丝绳最小破断载荷的  $80\%$  。如果钢丝绳绳端固定在升降机的驱动卷筒上，则卷筒上至少应保留两圈钢丝绳。

钢丝绳绳端应采用可靠的方法连接或固定（见图12），例如：

金属或树脂浇铸的接头；

带套环的编结接头；

带套环的压制接头；

楔形接头；

钢丝绳压板（可使钢丝绳在卷筒上有保留圈的钢丝绳固定装置）。

不应使用可能损害钢丝绳的绳端连接装置，如U形螺栓钢丝绳夹。

5.7.3.2.1.7 滑轮或卷筒节圆直径与钢丝绳直径之比不应小于30。

5.7.3.2.1.8 钢丝绳的储存应满足下列要求：

在张紧力下储存升降机接高备用钢丝绳时，应卷绕在带有螺旋绳槽的卷筒上。卷筒节圆直径与钢丝绳直径之比不应小于15。

如果储存的钢丝绳无张紧力，则可用无绳槽卷筒多层卷绕。释放储存钢丝绳张紧力的装置，应使被储存的钢丝绳的弯曲直径不小于钢丝绳直径的15倍。当钢丝绳绳夹压紧点之前的钢丝绳的张紧力，是在节圆直径不小于钢丝绳直径15倍的非转动卷筒上至少绕3圈来释放时，则因钢丝绳夹不会对钢丝绳造成损害而可使用。卷筒两端应有挡板，挡板外缘超出最外层钢丝绳的距离不应小于钢丝绳直径的2倍。

# 5.7.3.2.2 滑轮

滑轮应满足下列要求：

——绳槽应为弧形，槽底半径  $R$  应满足：  $1.05r \leq R \leq 1.075r$  ，其中  $r$  为钢丝绳半径；深度不应小于1.5倍钢丝绳直径；——引导钢丝绳上行的滑轮应防止异物进入；——应采取有效措施防止钢丝绳脱槽；——钢丝绳与垂直于滑轮轴的平面的夹角（钢丝绳偏角）不应大于  $2.5^{\circ}$  。

# 5.7.3.2.3 卷筒传动要求

钢丝绳应只绕一层。若使用自动绕绳系统，允许绕两层。留在卷筒上的钢丝绳不应小于两圈。卷筒两端应有挡板，挡板外缘超出最上层钢丝绳的距离不应小于钢丝绳直径的2倍。卷筒应有钢丝绳槽。钢丝绳与绳槽的偏斜角度（钢丝绳偏角）不应大于  $4^{\circ}$  。绳槽应满足下列要求：——绳槽的轮廓应为弧形，圆弧角不应小于  $120^{\circ}$  ，槽底半径  $R$  应满足：  $1.05r \leq R \leq 1.075r$  ，其中  $r$  为钢丝绳半径；——绳槽深度不应小于钢丝绳直径的1/3；——绳槽的节距不应小于1.15倍钢丝绳直径。

# 5.7.4 制动系统

5.7.4.1 每一吊笼都应设有制动系统，在下列情况下制动系统应自动动作：——主动力源中断；——控制回路失电。

5.7.4.2 制动系统中应至少有一个机- 电式制动器（摩擦式的），此外也可有其他制动措施（如电气的）。

5.7.4.3 不应使用带式制动器。

5.7.4.4 被制动作用的部件应与卷筒或驱动齿轮刚性连接。不应使用传动带或链条。

5.7.4.5 制动器应能使装有1.25倍额定载荷、以额定速度下行的吊笼停止。制动器也应能使装有额定载荷、以限速器动作速度运行的吊笼停止。在任何情况下，吊笼的平均减速度都不应大于  $1g$  。

5.7.4.6 制动器中产生制动作用力的任何弹簧，其制造和安装应满足：如果某一弹簧失效，载荷仍为额定载荷时，制动器仍可有效地使吊笼减速。

5.7.4.7 制动作用力应由压缩弹簧产生。压缩弹簧应被充分支撑，且其所受应力不应大于材料扭转弹性极限的  $80\%$  。

5.7.4.8 正常运行时，应有持续的电流来保持制动器的打开状态。

对于机- 电式制动器，切断制动器电流应由至少两个独立的电气装置单独实现，不论这些装置是否切断升降机的供电电流。

当升降机静止时，如果其中一个装置未切断制动器的电流，则最迟应在下一次运动方向改变时防止

吊笼有进一步的动作。

5.7.4.9 只要切断了对制动器的电流供应，制动器就应无延迟地动作（使用直接连接到制动线圈端子的二极管或电容器，不视为延迟方法）。

5.7.4.10 制动器应有表面磨损补偿调整措施。

5.7.4.11 制动器的防护等级至少应为GB/T4208中规定的IP23。

5.7.4.12 每一制动器都应能手动释放，且应用需用持续的作用力来维持释放状态。

# 5.7.5 对重

5.7.5.1 吊笼不应用作另一吊笼的对重。

5.7.5.2 对重的上下两端应设有合适的导靴或滚轮导向。

5.7.5.3 若对重使用填充物，则应采取措施防止其窜动。

应标明所需对重的总质量，每一单独填充物上应标明其自重。

5.7.5.4 对重应按GB2893的规定涂成安全色。

5.7.5.5 如果制造商允许升降机对重的下方有人可到达的空间，则对重应配备超速安全装置。

# 5.8 电气安装与电气设备

# 5.8.1 通则

电气安装和电气设备应符合GB/T5226.1的规定。

此外，对于电子器件，应考虑制造商规定的使用环境温度。当使用环境温度超出GB/T5226.1规定的温度限值时，应采取适当措施，例如加热或散热。

# 5.8.2 电气故障的防护

5.8.2.1 下列任何单一的升降机电气设备故障，不应导致升降机危险失灵。

a) 失压和无电压；b) 电压下降  $20\%$  以上；c) 对地或金属构件的绝缘破坏；d) 电气元件（如电阻器、电容器、晶体管、灯等）短路或断路以及参数或功能的改变；e) 接触器或继电器可动衔铁不吸合或不完全吸合；f) 接触器或继电器可动衔铁不分离；g) 触点不断开；h) 触点不闭合。

5.8.2.2 对于符合GB/T14048.5—2017中5.8.7要求的安全触点，不必考虑触点不断开的情况。

5.8.2.3 在电源错相或断相的情况下，机器应不能启动。

5.8.2.4 如果方向控制装置的电源断相，机器应停止运动，或者应至少不达到限速器的动作速度。

5.8.2.5 控制电路的设计应避免驱动电动机起发电作用而引起的危险情况。

5.8.2.6 电气安全装置的回路接地或与金属构件相连而接地，电气安全装置应立即停止机器的运动。应只有通过专业人员才能恢复其正常运行。

# 5.8.3 外界干扰防护

任何电气设备都应有保护以不受外部作用和落物（如雨、雪、砂浆、灰尘）的有害或危险影响。防护等级为：便携式控制装置不应低于GB/T4208中规定的IP65，控制柜、开关和制动器电气元件不应低于GB/T4208中规定的IP53，电动机不应低于GB/T4208中规定的IP44。

# 5.8.4 配线

升降机所有电缆和配线的布位和安装，应使其防止机械损伤。应特别注意吊笼上悬挂的电缆的强度和气候对其的影响。

为防止不正确的插接，所用的插头和插座都应有符合GB/T5226.1—2019中13.4.5要求的机械编码或等效标志。

# 5.8.5 接触器、继电器

交流或直流电动机的主接触器的使用类别不应低于GB/T14048.4规定的AC- 3或DC- 3。

用作主接触器的继电器，控制交流电磁铁的使用类别不应低于GB/T14048.5规定的AC- 15，控制直流电磁铁的使用类别不应低于GB/T14048.5规定的DC- 13。

在采取措施以满足5.8.2.1的过程中，主接触器和继电器都应：——只要有一个“常闭触点”闭合，则所有“常开触点”分离；且——只要有一个“常开触点”闭合，则所有“常闭触点”分离。

# 5.8.6 电气安全装置

5.8.6.1附录A列出的任一电气安全装置动作时，应按GB/T5226.1规定的0类停机功能要求，防止或立即停止机器的运动。电气安全装置应包括：

直接切断5.9.6.1中所述接触器供电的符合5.8.7要求的一个或多个安全触点；或符合GB/T7588.1—2020中5.11.2.3规定的安全链一部分的安全电路。

5.8.6.2 正常运行时，任何电气设备都不应与电气安全触点并联。

5.8.6.3电气安全装置的控制元件应能在连续正常运行产生的机械应力下正常工作。应不能用简单的方法使电气安全装置失效（用桥接件不视为简单的方法）。

5.8.6.4 安全装置开关的安装应按GB/T18831的要求进行。

# 5.8.7 安全触点

5.8.7.1安全触点应满足5.8.3的要求，并且其额定绝缘电压不应小于  $250~\mathrm{V}$  0

安全触点在交流电路中应符合GB/T14048.5中规定的AC- 15，在直流电路中应符合GB/T14048.5中规定的DC- 13。

5.8.7.2电气安全触点应按5.9.6的要求作用于机器的供电。

如果由于动力传输的原因，继电接触器被用于控制机器时，则应将其视为直接控制机器的动力供应以使机器启动或停止。

# 5.8.8 照明

只要升降机处于工作状态，吊笼内都应有照明。控制装置处的照明不应小于  $50\mathrm{lx}$

# 5.9 控制和限位装置

# 5.9.1 通则

所有的控制装置都应符合ISO9355- 1的要求。

# 5.9.2 行程限位开关

# 5.9.2.1 上、下行程开关

应设有上、下行程开关。上、下行程开关应使以额定速度运行的吊笼在接触到上、下极限开关前自

动停止。但不应以触发上行程开关作为最高层站停靠的通常操作。

# 5.9.2.2 上、下极限开关

5.9.2.2.1 在行程上端和下端均应设置极限开关，其应能在吊笼与其他机械式阻停装置（如缓冲器）接触前切断动力供应，使吊笼停止。5.9.2.2.2 极限开关动作后，应只有通过专业人员的干预才能使吊笼恢复运动。5.9.2.2.3 极限开关与行程开关不应共用一个触发元件。5.9.2.2.4 极限开关应符合5.8.6的要求。5.9.2.2.5 极限开关均应由吊笼或其相关部件的运动直接触发。

# 5.9.3 防松绳装置

钢丝绳式升降机和对重用的钢丝绳应设有防松绳装置，该装置应有符合5.8.6的松绳开关，钢丝绳松绳时其应中断吊笼的任何运动，直到经专业人员排除松绳后才能恢复吊笼运动（另见5.7.3.2.1.1）。

# 5.9.4 安装用附件

在升降机正常运行和安装、拆卸和维护/检查作业期间，任何安装用附件的正确位置，应由设计来确定，或者应由符合5.8.6的电气安全装置的核验来确定。安装用附件应包括导轨架提升装置、附墙架安装用延伸通道等。

# 5.9.5 停机装置

应设有使升降机(包括动力驱动的门)停机和保持非工作状态的停机装置。停机装置应设置在：

a）底部防护围栏外。b）滑轮间(如果有。滑轮间指仅装设滑轮或限速器和电气设备的房间)。c）吊笼顶，如果吊笼顶预定可使用。d）安装/拆卸/维护/检查控制装置中。e）吊笼内。a)和b)中的停机装置应为符合GB/T19670的停机装置，其功能应清楚标志。停机装置应包括符合5.8.6的电气安全装置。c）、d)和e)中的停机装置应为符合GB/T16754的急停装置。

# 5.9.6 停机

5.9.6.1 由电气安全装置动作引发的停机，应通过下列方式之一切断电动机的供电来实现：

由电气安全装置本身来完成；或由触点都串联在供电回路中的两个独立接触器来完成。

本文件没有包括接触器之外的其他停机装置的使用。如果能保证与本文件安全水平相同，也可使用其他装置，见GB/T7588.1。

5.9.6.2 升降机静止时，如果5.9.6.1提到的任一接触器的主触点没有打开，则最迟应在下一次运动方向改变时防止吊笼有进一步的动作。

# 5.9.7 控制方式

# 5.9.7.1 正常运行

5.9.7.1.1 在吊笼内应能对升降机进行控制。也可在地面和层站进行控制。

5.9.7.1.2 除急停外, 所有控制装置的设计都应使其只能通过有意的手动操作才能被驱动。5.9.7.1.3 应有防止吊笼在其停止后  $2 \mathrm{~s}$  之内离开层站的装置。

# 5.9.7.2 安装、拆卸和维护作业

5.9.7.2.1 安装、拆卸和维护/检查的控制操作应只能在吊笼内或吊笼顶进行。允许在  $1.1 \mathrm{~m}$  之上的侧面部分被拆除的吊笼内进行安装、拆卸和维护/检查作业, 但这些部分重新封围的正确结合应通过符合5.8.6的电气安全装置的核验, 以保证升降机的正常运行。

5.9.7.2.2 在安装、拆卸和维护/检查时, 吊笼最大速度不应大于  $0.7 \mathrm{~m} / \mathrm{s}$ , 并且除下列特殊情况外, 所有安全装置均应对吊笼的运行起保护作用:

——上极限开关和上行程开关可不起作用, 此时应采用自动越程保护措施, 如通过电气开关 (另见5.6.1); ——层门开关回路可短接。

5.9.7.2.3 安装、拆卸和维护/检查作业应设有控制装置。该装置应包括:

——符合5.8.6中有关电气安全装置要求的维护/检查开关, 该开关应是双稳态的, 应能锁住并不应受维护/检查控制位置以外的所有控制信号的干扰。应只有通过维护/检查开关的操作, 才能恢复升降机的正常运行; ——保持- 运行式控制装置, 其应设计成只能由有意的手动操作来触发, 并且其上应清楚地标明运行方向; ——符合5.9.5的急停装置。

# 5.10 故障状态

# 5.10.1 报警装置

为了便于吊笼乘员寻求外部援助, 应在吊笼内安装易于识别和接近的报警装置。

该装置应为铃声或类似装置或内部通信系统, 这些装置或系统应能在升降机断电后至少  $1 \mathrm{~h}$  内正常工作。

# 5.10.2 紧急逃离

乘员应可通过5.6.1.6所描述的方法从吊笼内逃出。

# 5.10.3 吊笼内被困人员的手动下降

如果吊笼内没有任何人都可操作的手动紧急下降装置, 则应符合下列要求: ——传动系统的制动器应可在吊笼内手动释放, 保持制动器打开所需的持续力不应大于  $400 \mathrm{~N}$ ; ——该装置应有保护以避免误用, 例如使用在紧急情况下能打破的保护盖; ——速度应自动控制且应小于限速器的动作速度, 也不应大于  $1.0 \mathrm{~m} / \mathrm{s}$  。

# 5.10.4 专业人员的紧急操作

# 5.10.4.1 通则

未配备符合5.10.3规定的手动紧急下降装置的升降机, 应有仅供专业人员使用的紧急操作措施, 以在紧急情况时将吊笼移动到层站。

# 5.10.4.2 人工紧急操作

移动装有额定载荷的吊笼所需的手动操作力不应大于  $400 \mathrm{~N}$  。

该措施应只有专业人员可接近。

# 5.10.4.3 电气紧急操作

在设有电气紧急操作的地方，应安装符合5.8.6要求的紧急操作开关。该开关应只有专业人员可接近，以防止吊笼的所有运动，但由该开关控制的除外。

紧急操作开关应允许通过保持- 运行式控制装置来控制吊笼的移动，该控制装置的设计应使其只能通过有意的手动操作来驱动，并应设置在紧急操作开关附近。应在上下按钮附近清楚地标明运动方向。

当吊笼被提升时，允许紧急操作开关直接或通过其他安全装置使在回路中的限速器、超速安全装置、缓冲器、极限开关和松绳开关所要求的电气安全装置失效。

紧急操作开关及其按钮的位置，应使其在使用该开关时易于观察到吊笼的运动。

吊笼的速度不应大于  $0.7\mathrm{m / s}$

# 5.11 噪声

# 5.11.1 通则

机器的设计和制造应考虑降低噪声(特别是降低声源噪声)的技术进步和措施的可用性，将空气噪声发射产生的风险降至最低水平。噪声发射级可参照类似机器的可比较发射数据进行评估。

# 5.11.2 在设计阶段减小噪声

设计机器时，应考虑GB/T25078.1中给出的控制声源的信息和技术措施。也可考虑GB/T25078.2中给出的信息。最大的噪声源是指枪齿条传动装置。

# 5.11.3 噪声的测量方法

底部防护围栏（距底部防护围栏外  $1.00\mathrm{m}$  、离地高  $1.60\mathrm{m}$  处)和吊笼内工作位置处的发射声压级，应根据GB/T17248.2—2018中的5.2.2采用精度等级2的工程法测量。

噪声测量期间的运行工况：空载。

噪声声明应符合7.1.2.3的要求。

# 6.2 特别验证试验

# 6.2.1 通则

6.2.1.1 应对下列部件进行特别验证试验：

层门和吊笼门的门锁装置；超速安全装置和限速器；耗能型缓冲器和线性蓄能型缓冲器。应编制和保存试验记录。

6.2.1.2 试验应由具备相应资质的试验机构或具备试验条件的升降机制造商、零部件制造商来进行。

仪器的精度应在下列范围内：质量、力、距离、时间、速度：  $\pm 2\%$  重加速度、减速度：  $\pm 4\%$  电压、电流：  $\pm 2\%$  温度：  $\pm 2^{\circ}C$  0

6.2.1.3 报告应至少包括以下内容：

试验设备的名称；试验报告编号；类别、型式和产品名称（或商品名称）；制造商名称和地址；试验日期；符合6.2.2.1.3要求的文件；地点、时间、试验者姓名。

# 6.2.2 吊笼门和层门的门锁装置

# 6.2.2.1 通则

# 6.2.2.1.1 应用范围

本程序适用于门锁装置。所有参与门的锁紧和检查锁紧状态的零部件均为门锁装置的组成部分。

# 6.2.2.1.2 试验目的

应按本试验程序检验门锁装置的结构和动作是否符合本文件。

# 6.2.2.1.3 相关文件

试验报告中应包括下列文件：

带有工作原理说明的结构示意图，应清楚地表明与门锁装置工作和安全相关的所有细节，包括锁紧元件的有效接合以及电气安全装置的作用位置以及紧急开锁装置的操作；门锁装置的类型（AC和/或DC）及其额定电压和额定电流的说明。

# 6.2.2.1.4 试验样品

应在样品上进行试验。

如果门锁装置的试验只能在将其安装于相应的门（如有几块门板的滑动门或铰链门）上时才能进行，则应按工作状况将门锁装置安装在完整的门上。在不影响试验效果的条件下，门的尺寸可比真实门的尺寸小。

# 6.2.2.2 检查和试验

# 6.2.2.2.1 动作检查

试验的目的是验证门锁装置的机械和电气元件是否能按安全要求正确动作，是否符合本文件的要求。

尤其应对下列事项进行验证：

在电气安全装置动作之前，锁紧元件至少有  $7\mathrm{mm}$  的接合长度（见5.5.5.3.9）；

当门打开或未锁紧时，人在正常可接近的位置不能使升降机运行。

# 6.2.2.2.2 试验

# 6.2.2.2.2.1 通则

试验的目的是验证门锁装置的机械元件和电气元件的强度。处于正常工作位置的门锁装置样品，应控制其按正常使用情况进行动作。门锁装置样品应按其制造商的常规要求进行润滑。动作循环次数应由机械或电子计数器记录。

# 6.2.2.2.2.2 耐久性试验

门锁装置应进行200000次  $(\pm 1\%)$  完整循环（一次循环包括一次开锁作业和一次重新锁紧作业）。门锁装置的驱动应平滑无冲击。在耐久性试验中，电气安全开关应通过锁按正常工作状态进行机械动作。

# 6.2.2.2.2.3 静态试验

试验时应施加  $1\mathrm{kN}$  的静力，且总持续时间不应小于  $5\mathrm{min}$  。该力应作用在门打开的方向，且尽量接近使用者试图打开门时力所作用的位置。

# 6.2.2.2.2.4 试验准则

耐久性试验(6.2.2.2.2)和静态试验(6.2.2.2.2.3)后，不应出现任何影响安全性的磨损、变形或损坏。

# 6.2.3 超速安全装置和限速器

# 6.2.3.1 通则

利用正常工作工况时使用的悬挂和导向系统进行试验，超速安全装置应与其限速器联合起来一起进行试验。也可采用经过验证是正确的其他方法进行试验。

制造商应说明试验时的载荷质量  $(\mathrm{kg})$  和额定速度  $(\mathrm{m / s})$  。如果超速安全装置要在不同的载荷质量和速度下进行验证，应详细说明。

# 6.2.3.2 试验方法

6.2.3.2.1 应对每类新型的超速安全装置及其相应的限速器的典型结合进行试验，坠落试验的基本载荷为所有与吊笼一起运行的质量的总和。

6.2.3.2.2 试验应在制造商规定的限速器动作速度下进行。

6.2.3.2.3 应按下列规定之一进行试验：

a）在传动装置正常连接和脱开的情况下进行试验，试验次数不应少于30次，并至少进行下列试验：

- 吊笼空载、传动装置正常连接，试验5次；- 吊笼载有额定载荷、传动装置正常连接，试验10次；- 吊笼载有1.3倍额定载荷、传动装置正常连接，试验5次；- 吊笼载有额定载荷、脱开传动装置的驱动连接，试验5次；- 吊笼载有1.3倍额定载荷、脱开传动装置的驱动连接，试验5次。

b）在传动装置正常连接的情况下进行试验，试验次数不应少于40次，并至少进行下列试验：

- 吊笼空载，试验5次；

- 吊笼载有额定载荷, 试验 20 次;- 吊笼载有 1.3 倍额定载荷, 试验 15 次。

6.2.3.2.4 直接或间接测量应包括：

a) 下落总高度，b) 制动距离，c) 动作速度，d) 以时间为函数的减速度。

6.2.3.2.5 应对下列项目进行检查：

6.2.3.2.5 应对下列项目进行检查：a) 制动距离和/或减速度是否符合 5.6.2.3，b) 最大动作速度是否符合 5.6.2.12，c) 有无裂纹和变形，d) 由钢丝绳驱动的限速器, 限速器动作时作用在钢丝绳上张力是否符合 5.6.2.14。

# 6.2.3.3 试验报告

报告应载明：

6.2.3.3 试验报告报告应载明：a) 6.2.1.3 规定的内容；b) 超速安全装置和限速器的型式和适用性；c) 升降机的类型和型式；d) 包括惯性效应在内, 超速安全装置制停的总载荷；e) 限速器动作速度；f) 对于超速安全装置运行的导轨, 导轨的许用厚度、夹紧面的最小宽度、导轨润滑和导轨表面情况；g) 由钢丝绳驱动的限速器, 其钢丝绳的直径、结构, 限速器触发时作用在钢丝绳上的张力, 如有牵引滑轮, 则是最小张力。

# 6.2.4 耗能型缓冲器和线性蓄能型缓冲器

缓冲器的特别型式验证试验，应按GB/T7588.2—2020中5.5的型式试验程序进行。

# 6.3 在第一次使用之前对每台升降机的验证试验

制造商应进行升降机的静载和动载试验，以保证升降机正确制造和装配，核验所提供的全部装置是否齐备并正确工作。这些试验可在制造商或其授权代表处或在使用地点进行。

尤其应验证并在试验报告中报告下列项目：- 所有限位开关的正确功能，- 所有控制装置的正确功能，- 在规定限值内超速安全装置的功能，- 在  $1 \sim 1.1$  倍额定载荷下超载检测装置的动作，- 在规定限值内制动系统制动距离，- 门锁装置的正确功能，- 1.1 倍额定载荷的动载试验，- 1.25 倍额定载荷的静载试验，- GB/T 5226.1 规定的电气试验。

# 7.1.2.8.2 净距

# 7.1.2.8.2.1 升降机周围的净距

升降机周围的间距应符合下列要求：

如果任一通道点与其相邻的升降机任一运动件之间的安全距离小于  $0.85\mathrm{m}$  （额定速度不大于 $0.7\mathrm{m / s}$  时，为小于  $0.5\mathrm{m}$  )，则升降通道防护装置应符合GB/T23821—2009中表1和表4的要求，且高度不应小于  $2.0m$  ，或者为地板到天花板的全高度，此时可小于  $2\mathrm{m}$  0

如果安全距离不小于  $0.85\mathrm{m}$  （额定速度不大于  $0.7\mathrm{m / s}$  时，为不小于  $0.5\mathrm{m}$  )，则应配备高度不小于 $1.1\mathrm{m}$  的固定式防护装置（有扶手、中间横杆和护脚板）。

# 7.1.2.8.2.2 吊笼上方的净距

为预防吊笼在升降通道的顶端超出行程限度，应有充分的导向高度作为越程余量。对于额定速度不大于  $0.85\mathrm{m / s}$  的升降机，其越程余量应为：

对重质量大于空吊笼质量的升降机：  $\geq 2\mathrm{m}$  钢丝绳式或对重质量不大于空吊笼质量的升降机：  $\geq 0.5\mathrm{m}$  齿轮齿条式升降机：  $\geq 0.15\mathrm{m}$  0

越程余量应从上极限开关的动作触点算起；对于带对重的升降机，越程余量应从对重与其缓冲器的接触点算起。

如果额定速度大于  $0.85\mathrm{m / s}$  ，越程余量应增加  $0.1v^{2}(m)$  ，其中  $\mathcal{V}$  为额定速度  $(m / s)$  0

当吊笼运行到达越程余量终点时，其上还应有不小于  $1.8m$  （如果吊笼顶部不允许上人，则应有至少  $0.3\mathrm{m}$  )的自由净空距离。此外，吊笼上任何高过吊笼的部件和设备，其上方应有不小于  $0.3\mathrm{m}$  的自由净空距离。

# 7.1.2.8.2.3 对重上方的净距

当吊笼停在充分压缩的缓冲器上时，对重上方应有不小于  $0.3\mathrm{m}$  的自由净空距离。

# 7.1.2.8.3 基础

应给出能承受所有载荷的升降机用场地的信息。支承底架和导轨架的基础，应能承受5.2描述的力和力矩。

如升降机底部的下方有可接近的空间，则任何对重都应配有防坠安全装置。

# 7.1.2.8.4 安装和拆卸

7.1.2.8.4 安装和拆卸应只有专业人员才能安装和拆卸升降机。只应由专业人员来进行升降机的供电连接。使用说明书应至少包括：推荐使用的剩余电流保护器（漏电保护器）；往工地的运输；

升降机的额定安装载荷和额定安装速度；- 导轨架节和附墙架的安装，包括螺栓（直径、性能等级、拧紧力矩）的正确使用信息；- 提升较重部件的建议方法；- 底部防护围栏和层门的安装和安全要求，包括每一通道点都应有的升降通道防护装置；- 层站照明（说明应包括：如使用升降机时照明不足，则应提供充足的土地照明，照亮升降机全运行高度上的层站）；- 整机试验；- 拆卸程序。

整机试验应至少包括下列要求：

升降机的核对检查；

功能试验：

- 驱动系统和制动器，- 行程限位，- 吊笼和层门，- 超载检测装置，- 净距和间距，- 升降机应在吊笼装有制造商规定的额定载荷下进行全行程的动态试验，- 吊笼防坠安全装置应进行制造商规定的动态试验。

# 7.1.2.9 操作和使用说明

使用说明书中应有单独的章节，向用户提供安全操作和操作人员培训的最低要求的明确信息。

对于升降机的操作，应明确只能由专职操作者(司机)操作还是被准许进入工地的人员都可操作。如要求只有专职操作者(司机)才能操作，还应明确其职责，另见5.5.5.1。被准许进入工地利用升降机进行施工作业的人员应都可进行升降机的装载、卸载作业。

详细说明应包括：

- 预定使用；- 层门和吊笼门的操作；- 吊笼的装载以及对装载位置、载荷集中堆放和载荷固定的限制；- 推行带小脚轮的物品器具进出吊笼时的适当间隙；- 正常运行工况下吊笼顶应无载荷；- 吊笼内、底部防护围栏和层站可使用的控制装置的功能；- 环境条件，如工作状态最大风速。

# 7.1.2.10 紧急情况处置程序

使用说明书中应有单独的章节，向专业人员提供紧急情况处置的全部必要信息，如：

- 特设控制器；- 安全装置，如限位开关、超速安全装置；- 故障的处置；- 回路图；- 关于通往吊笼的通道信息（见5.6.1.5.1.9和5.6.1.6）。

# 7.1.2.11 定期检查和维护

使用说明书中应说明按制造商要求、运出工况和使用频率而确定的定期检查、试验和维护的频次，

并应给出检查细目和适用性的详细信息。

如未随机器提供运行记录本，则还应在使用说明书中说明运行记录内容。

使用说明书中还应说明易损件及其更换标准，如GB/T5972有关钢丝绳的更换。使用说明书应有关于彻底检查疲劳寿命（见5.2.6.1)的章节。

# 7.2 标志

# 7.2.1 通则

7.2.1 通则按GB/T15706—2012中6.4.4的规定，制造商应将标有  $7.2.2\sim 7.2.8$  内容的标牌或标志固定在升降机和其部件的明显部位。标牌或标志应持久耐用。

# 7.2.2 标牌

7.2.2 标牌应有下列信息：- 制造商或授权代理商的名称、地址；- 型式；- 序列号或产品编号；- 生产日期；- 额定载荷（正常作业工况、安装/拆卸工况），单位为千克（kg）；- 提升高度；- 钢丝绳参数；- 基本单元的质量，单位为千克（kg）；- 额定速度（正常作业工况、安装/拆卸/维护/检查工况）。

# 7.2.3 导轨架节或导轨节的识别标志

每个导轨架节或导轨节上都应有可识别其生产日期的标志或序号。

# 7.2.4 基本使用信息标牌

应有下列信息：

7.2.4 基本使用信息标牌应有下列信息：- 导轨架高度；- 导轨架顶端自由高度；- 导轨架节连接螺栓（螺栓直径、性能等级、拧紧力矩）；- 电力供应（包括剩余电流保护器的参数）；- 自由独立时（未与地层锚固也未附墙）的操作；- 安全距离；- 附墙架示意图，包括附墙间距；- 非工作状态下吊笼的位置；- 操作说明书的查看要求；- 有关对重下方有可接近的空间时，在该空间的上方设置的警示信息。

# 7.2.5 吊笼内的标牌

7.2.5 吊笼内的标牌应有字符高度不小于  $25\mathrm{mm}$  的下列信息：- 正常运行工况的额定载荷和最大乘员数；- 对载荷位置和集中堆放的限制；

GB/T 26557—2021

——本升降机只允许专职操作者(司机)操作(如果有此要求)。

# 7.2.6 底部防护围栏出入口外的标牌

应有下列内容：——非经许可严禁入内。

# 7.2.7 超速装置标牌

7.2.7 超速装置标牌应有下列信息：——制造商名称和地址；——型式检验证书编号；——动作速度；——生产日期和产品编号或序列号。

# 7.2.8 驱动电动机标牌

7.2.8 驱动电动机标牌应有下列信息：——制造商名称和地址；——型式；——生产日期和产品编号或序列号。

# 7.3 控制元件标志

所有的控制元件都应清楚地标志。推荐使用图形符号。
