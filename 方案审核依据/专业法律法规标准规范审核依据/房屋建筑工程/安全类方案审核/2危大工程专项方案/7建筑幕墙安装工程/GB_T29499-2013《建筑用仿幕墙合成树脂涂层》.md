本标准由中华人民共和国住房和城乡建设部提出。

本标准由全国建筑构配件标准化技术委员会(SAC/TC454)归口。

本标准起草单位:深圳市嘉达高科产业发展有限公司、国家化学建筑材料测试中心、合众(佛山)化工有限公司、深圳市嘉达节能环保科技有限公司、深圳市标准技术研究院、陕西理工学院。

# 建筑用仿幕墙合成树脂涂层

# 1范围

本标准规定了建筑用仿幕墙合成树脂涂层的术语和定义、产品分类、基本构造、材料、要求、试验方法、检验规则、标志、包装和贮存。

本标准适用于由腻子、底漆和涂料组成，具有幕墙外观效果的建筑用涂层的要求和检验。

# 3术语和定义

下列术语和定义适用于本文件。

3.1

仿幕墙合成树脂涂层 synthetic resin curtain- wall- imitation coating

以合成树脂为主要粘结材料,与颜料、体质颜料及各种助剂配制成各种腻子以及涂料,分层施涂在建筑物墙体上,形成具有幕墙外观的建筑装饰层。

3.2

质感层 texture coat

采用厚质涂料在底涂层上形成具有特定质感的涂层。

# 4产品分类

本标准根据仿幕墙合成树脂涂层面层涂料的不同，分为两种类型：

a）I型面层涂料为氟树脂涂料；

b）Ⅱ型面层涂料为其他类型涂料

# 8 试验方法

# 8.1 试验环境

应按GB/T9278的规定进行调节。

# 8.2 试验基材的制备

8.2.1 无石棉纤维水泥平板

无石棉纤维水泥平板应符合JC/T412.1- 2006中NAFHV级的技术要求,厚度为4mm~6mm,表面处理应按GB/T9278的规定进行。

# 8.2.2 混凝土墙

# 8.2.3 砂浆块

# 8.3 外观

# 8.3.1 试样制备

要求如下:

a) 仿幕墙合成树脂涂层基本构造一的制备工艺:

1) 将找平腻子批压在8.2.1规定的试板表面,厚度为  $1.5 \mathrm{~mm}$  。腻子的批压应使用钢制刮板(或刮刀),用力反复批压,确保腻子层密实、表面平整、无残留气泡。

2) 找平腻子未干时铺贴耐碱玻璃纤维网布,再批压1遍找平腻子,厚度为  $1.5 \mathrm{~mm}$  应完全覆盖住耐碱玻璃纤维网布。在标准环境下养护7d。

3) 批压柔性腻子,厚度为  $1 \mathrm{~mm}$  。在标准环境下养护7d。

4) 批压抛光腻子,厚度为  $0.5 \mathrm{~mm}$  。12h后使用320目干磨砂纸进行手工打磨,用软布除去打磨产生的粉尘。在标准环境下养护7d。

5) 应按JG/T210-2007中6.4.2的规定刷涂底漆。在标准环境下养护24h。

6) 喷涂2道涂料,每道间隔24h,每道厚度为  $(25 \pm 5) \mu \mathrm{m}$  。在标准环境下养护28d。

b) 仿幕墙合成树脂涂层基本构造二的制备工艺:

1) 应按8.3.1a)1)和按8.3.1a)2)的方法批压找平腻子和铺贴耐碱玻璃纤维网布。

2) 应按JG/T210-2007中6.4.2的规定刷涂底漆。在标准环境下养护24h。

3) 批压或喷涂质感层,并按厂家提供的方法进行处理以获得浮雕等效果造型,最薄处厚度为  $1 \mathrm{~mm}$  最厚处厚度为  $3 \mathrm{~mm}$  。在标准环境下养护48h。

4) 喷涂2道涂料,每道间隔24h,每道厚度为  $(25 \pm 5) \mu \mathrm{m}$  。在标准环境下养护28d。

# 8.3.2 试验方法

将制备好的样板一边呈水平方向,板面与水平面成约  $85^{\circ}$  角竖放。在非阳光直射的自然条件下进行试验,距离板面约  $1000 \mathrm{~mm}$  目视观察。

# 8.4 耐水性

# 8.4.1 试件制备

在试验基材上按8.3.1的工艺制备试板。批压腻子时可借助相应尺寸和厚度的型框。用  $1:1$  的松香和石蜡混合物将试板四周边缘和背面封闭,封边宽度  $2 \mathrm{~mm} \sim 4 \mathrm{~mm}$  。

# 8.4.2 试验步骤

应按GB/T1733- 1993甲法规定进行。试板投试前除封边外,还应封背。将3块试板浸入GB/T6682规定的三级水中。

8.5 耐碱性

8.5.1 试件制备

应按8.3.1制备试件。

8.5.2 试验步骤

8.5.2 试验步骤应按GB/T9265规定进行。

8.6 耐候性

8.6.1 试件制备

8.6.1 试件制备在8.2.2规定的混凝土墙上应按8.3.1的工艺制备试板。批压腻子时可借助相应尺寸和厚度的型框。

8.6.2 试验步骤

步骤如下:

a)热雨雨高温-冰水循环。试板应依次经过下列步骤进行80次"高温-冰水循环",每次6h,

1)加热3h:将试板表面温度升至70℃并恒温在(70±5)℃,恒温时间不应小于1h;恒温过程中耐候性试验箱内空气相对湿度不应高于30%

2)喷水1h:向试件表面喷水,水温(16±5)℃,喷水量1.0L/(m²·min)~1.5L/(m²·min)。

3)静置2h:

b)状态调节。

在温度10℃~20℃,相对湿度不小于50%的条件下应需45h

c)热冷周期加热-冷冻循环。试板应依次经过下列步骤进行20次"加热-冷冻循环",每次24h。

1)将试板表面加热至50℃(温度上升时间为1h),保持温度(50±5)℃,相对湿度不大于10%恒温时间7h。

2)将试板表面降温至-20℃(温度下降时间为2h),保持温度(-20±5)℃恒温时间14h。

3)加热8h:将试件表面温度升至50℃并恒温在(50±5)℃,恒温时间不应小于5h;恒温过程中耐候性试验箱内空气相对湿度不应高于30%

4)制冷16h:将试件表面温度降至-20℃并恒温在(-20±5)℃,恒温时间不应小于12h。

8.7 耐冻融

8.7.1 试件制备

8.7.1 试件制备在8.2.3规定的砂浆块上应按8.3.1的工艺制备试板。批压腻子时可借助相应尺寸和厚度的型框。用1:1的松香和石蜡混合物封边,封边宽度2mm~4mm。

8.7.2 试验步骤

8.7.2 试验步骤在(23±2)℃水中浸泡8h,(-20±2)℃冰箱中冷冻16h为1次循环,应做30次循环。

8.8 耐人工气候老化性

8.8.1 试件制备

8.8.1 试件制备在试验基材上应按8.3.1的工艺制备试板。批压腻子时可借助相应尺寸和厚度的型框。

# 8.8.2 试验步骤

应按GB/T1865规定执行。

# 8.9 耐冲击性

# 8.9.1 试件制备

在8.2.2规定的混凝土墙上应按8.3.1的工艺制备试板,批压腻子时可借助相应尺寸和厚度的型框。

# 8.9.2 试验步骤

将试板置于厚度为  $20 \mathrm{mm}$  的符合GB/T17671要求的标准砂上面,有涂层的一面朝上,然后把直径  $(50 \pm 2) \mathrm{mm}$ ,质量为  $(500 \pm 10) \mathrm{g}$  的钢球从高度为  $500 \mathrm{mm}$  处自由落下,用肉眼观察试件表面。在试板上选择各相距  $50 \mathrm{mm}$  的3个位置进行。

# 8.10 耐沾污性

# 8.10.1 试件制备

在试验基材上应按8.3.1的工艺制备试板。批压腻子时可借助相应尺寸和厚度的型框。

# 8.10.2 试验步骤

应按GB/T9780规定进行。

# 8.11 拉伸粘结强度

# 8.11.1 试板的制备

在8.2.3规定的砂浆块上应按8.3.1的工艺制备试板。如为基本构造二,喷涂质感层后应用刮板平整表面,质感层厚度为  $3 \mathrm{mm}$ 。

# 8.11.2 试验步骤

用适当的胶粘剂将试板上下表面分别与尺寸为  $100 \mathrm{mm} \times 100 \mathrm{mm}$  的金属试验板粘结。通过万向接头将试板安装于拉力试验机上,拉伸速度为  $5 \mathrm{mm} / \mathrm{min}$ ,拉伸至破坏标记录破坏时的拉力及破坏部位。

# 8.11.3 结果计算

当试板的破坏部位在粘结界面处,则此试板的试验数据无效。有效试验数据不应少于5个。

在有效试验数据中去掉1个最大值和1个最小值,取剩余数据的算术平均值作为结果,各试验数据与平均值的最大相对偏差不应大于  $20\%$  ,否则本次试验无效。

# 9 检验规则

# 9.1 检验项目

9.1.1 出厂检验项目为:外观。

9.1.2 型式检验项目包括第7章规定的全部技术要求。有下列情形之一者,应进行型式检验:

a) 新产品或老产品转厂生产的试制定型鉴定;

b) 正式生产后,当结构、材料、工艺有较大改变而可能影响产品性能时;

c) 正常生产时每两年检验1次;

d) 产品停产两年后,恢复生产时;

e) 交收检验结果与上次型式检验有较大差别时;

f) 国家质量监督机构提出进行型式检验要求时。

9.2 组批

9.2 组批同一种类、同一级别、同一规格产品  $3000 \mathrm{~m}^{2}$  为1批,不足  $3000 \mathrm{~m}^{2}$  也作为1批。

9.3 判定规则

# 10 标志、包装和贮存

# 10.1 标志

10.1.1 找平腻子、柔性腻子、抛光腻子、底漆、质感层和涂料应按 GB/T 9750 的规定进行标志。对于双组分配套组成的产品,应明确各组分配比。如需稀释,应明确稀释剂和稀释比例。

10.1.2 耐碱玻璃纤维网布应按 JC/T 841- 2007 中 7.1 的规定进行标志。

# 10.2 包装

10.2.1 找平腻子、柔性腻子、抛光腻子、质感层和底漆应按 GB/T 13491 中二级包装要求的规定进行包装。

10.2.2 耐碱玻璃纤维网布应紧密整齐地卷在硬质纸管上,不应有折叠和不均匀等现象,用结实的防水防潮材料包装。

10.2.3 涂料应按 GB/T 13491 中一级包装要求的规定进行。

# 10.3 贮存

产品贮存时应保证通风、干燥,防止日光直接照射,冬季应采取适当防冻措施。溶剂型产品应按危险品贮存。根据产品类型分别定出贮存期,并在包装标志上明示。
