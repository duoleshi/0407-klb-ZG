# 装配式混凝土箱梁桥预算定额

# 1范围

本标准规定了装配式混凝土箱梁桥预算定额的总则、定额使用说明及预算定额。

本标准适用于公路装配式混凝土箱梁桥新建、改建工程，是编制初步设计概算、施工图设计预算的依据。

# 2规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

# 3总则

3.1使用装配式混凝土箱梁桥预算定额编制概算、预算应符合JTG3830和本标准的规定。

3.2本标准包括墩（台）预制混凝土、墩（台）预制钢筋、墩身预应力钢筋、墩（台）身存放、盖梁预制、横向预应力、构件运输、安装、注浆等九个分项工程。

# 4定额使用说明

4.1本标准是以人工、材料、机械台班消耗量表现的工程预算定额。编制预算时，其人工费、材料费、机械使用费、措施费、企业管理费、规费、利润、税金和专项费用，应按JTG3830的规定计算。

4.2本标准是按照合理的施工组织和正常的施工条件编制的，定额中采用的施工方法和工程质量标准根据DB41/T1847取定。除定额中规定允许换算外，均不得因具体工程的施工组织、操作方法和材料消耗与定额的规定不同而调整定额。

4.3 定额工作按每工日8h计算。

4.4本标准是以人工、材料、机械台班消耗量表现的公路工程预算定额。定额基价按JTG/T3832中有关人工、材料、机械台班单价附录计算，JTG/T3832未包含的材料按本标准附录A“装配式混凝土箱梁桥预算定额新增材料代号及材料单位重、损耗表”计算，JTG/T3833未包含的机械台班单价按本标准附录B“装配式混凝土箱梁桥预算定额新增机械台班费用定额”计算。

4.5混凝土强度等级或水泥强度等级与定额所列强度等级不同时，按配合比表进行换算。如采用自密实混凝土，其材料用量按附录C“装配式混凝土箱梁桥预算定额自密实混凝土配合比表”计算。

4.6编制公路专业装配式混凝土箱梁桥工程概算、预算时与JTG3830配套使用。

4.7 本标准中不包含水泥砼的配运料、拌合、运输，按 JTG/T 3832 规定计算。4.8 本标准中预制砼构件已包含平面底座摊销，场站建设另行计算。4.9 本标准未规定的按 JTG 3830 执行。4.10 计算规则按 JTG/T 3832 执行。
