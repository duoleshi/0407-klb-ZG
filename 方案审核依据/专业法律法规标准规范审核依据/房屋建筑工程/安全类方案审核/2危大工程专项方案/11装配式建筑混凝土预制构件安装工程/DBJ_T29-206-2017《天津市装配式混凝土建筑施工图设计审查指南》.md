# 编制总说明

为统一我市装配式混凝土建筑施工图设计审查标准、保证施工图设计质量，结合今后一段时期我市装配式建筑建设实际需要，根据《市建委关于开展装配式建筑施工图设计审查技术文件编制工作的通知》（津建设函[2016]227号）文件要求，天津市建设工程技术研究所组织天津市房屋鉴定建筑设计院完成《装配式混凝土建筑施工图设计审查指南》（以下简称审查指南）编制工作，经市建委批准为天津市工程建设标准设计技术文件。

本审查指南仅编入与装配式混凝土建筑有关的标准强制性条文和影响公共安全、公共利益的非强制性条文，以及与装配式混凝土建筑设计和审查有关注意事项。为高标准的推进我市装配式混凝土建筑施工图设计质量，本审查指南在参考国家标准基础上，将部分“宜”类条款按照“应”的要求列出，适当提高了我市施工图审查标准要求。我市装配式混凝土建筑工程除应按本指南的所列条目重点审查外，还应按《天津市民用建筑施工图设计审查要点》审查。

本审查指南版权归天津市建设工程技术研究所享有，编制单位享有署名权，未经允许，任何单位和个人无权转让。审查指南使用过程中如发现问题或有意见和建议，请与天津市建设工程技术研究所联系，以便修订和更新。

本审查指南编制过程中得到了有关部门领导和专家的大力支持，并提出许多宝贵意见和建议，在此一并感谢。

# 一、总则

1.1 为指导和规范天津市装配式混凝土建筑工程施工图设计文件的审查工作，根据国家及天津市相关法律法规、装配式建筑有关政策文件和《装配整体式混凝土剪力墙结构设计规程》DB/T 29- 244- 2016（以下简称DB/T 29- 244- 2016）、《装配式混凝土结构技术规程》JGJ 1- 2014（以下简称JGJ 1- 2014）、《装配式混凝土建筑技术标准》GB/T 51231- 2016(以下简称GB/T 51231- 2016)等标准编制本指南。

1.2 装配式混凝土建筑设计除应满足国家及天津市现行有关装配式标准规定外，还应符合国家及天津市现行相关标准的规定。装配式混凝土建筑工程施工图设计文件的审查，除应符合本指南的要求外，还应符合《天津市民用建筑施工图设计审查要点》的要求。

1.3 本指南适用于天津市抗震设防类别为标准设防类、重点设防类，抗震设防烈度为7度(0.15g)及8度(0.20g)的装配式混凝土建筑工程施工图设计文件的技术审查。

1.4 本指南的装配式混凝土结构包括：装配整体式框架结构、高层装配整体式剪力墙结构、多层装配式剪力墙结构、装配整体式框架- 现浇剪力墙结构、装配整体式框架- 现浇核心筒结构、装配整体式部分框支剪力墙结构。

1.5 当房屋高度、规则性、结构类型、结构装配方案和预制构件连接类型等超过DB/T 29- 244- 2016、JGJ 1- 2014、GB/T 51231- 2016的规定时，应进行专门研究和论证。

1.6 本指南列入了《装配式混凝土结构技术规程》JGJ 1- 2014中的强制性条文。其它工程建设标准中的强制性条文，本指南从略。本指南列入的工程建设标准一般性条文原则上是与结构安全及工程建设标准强制性条文有关的条文。本指南所列审查内容是保证工程设计质量的基本要求，并不是工程设计的全部内容。设计单位和设计人员应全面执行工程建设标准和法规的有关规定。

1.7 报审的装配式混凝土建筑工程施工图设计文件中应包含装配式建筑建设规模、预制装配率等指标以及预制装配率计算书等。装配式建筑建设规模应符合土地出让、项目立项、规划批复等文件的相关要求。

1.8装配式混凝土建筑工程预制装配率计算应执行《市建委关于印发<天津市装配整体式建筑预制装配率计算细则（试行）>的通知》（津建科〔2016〕464号）文件及国家相关标准的要求。

注：目前，天津市装配式建筑预制装配率计算执行津建科〔2016〕464号文件要求。如后续天津市有新版相关法规、政策文件和工程建设标准实施，应以新版法规、政策文件和工程建设标准为准。

1.9装配式混凝土建筑面积的计算应执行《市建委等七部门联合印发关于加快推进我市建筑产业现代化发展（2015- 2017年）实施意见的通知》（津建科〔2015〕543号）文件的要求。

注：目前，天津市装配式混凝土建筑面积的计算执行津建科〔2015〕543号文件要求，如后续天津市有新版相关法规、政策文件和工程建设标准实施，应以新版法规、政策文件和工程建设标准为准。

1.10报审的装配式混凝土建筑应提供涵盖各专业的装配式建筑设计专篇。各专业应在本专业施工图中提供各自的装配式专项设计说明。

1.11报审的装配式混凝土建筑工程结构专业设计图纸包括结构施工图和预制构件制作详图。结构施工图除应满足计算和构造要求外，其设计内容和深度还应满足预制构件制作详图编制和安装施工的要求。本指南内容仅涉及装配式混凝土结构施工图设计文件审查。对于报审的预制构件制作详图应满足生产的要求，根据本指南对预制构件详图属于结构施工图的内容进行技术审查。

1.12本审查指南中，“要点"所列内容为引用的国家及天津市现行标准及相关政策文件。“要求"所列内容为参照国家现行标准，结合天津市具体要求适当调整或提高标准后的要求。本审查指南中，黑体字为引用规范的强制性条文；宋体字为引用规范、标准和规定的非强制性条文；楷体字为对规范条文附加的说明。

1.13本指南发布后，如有新版相关法规和工程建设标准实施，应以新版法规和工程建设标准为准。

2给水分水器与用水器具的管道接口应一对一连接，在架空层或吊顶内敷设时，中间不得有连接配件，分水器设置位置应便于检修，并宜有排水措施；
3宜采用装配式管线及其配件连接；
4敷设在吊顶或楼地面架空层的给水管道应采取防腐蚀、隔声减噪和防结露等措施。
7.2.4装配式混凝土建筑的太阳能热水系统应与建筑一体化设计。
7.2.5装配式混凝土建筑应选用耐腐蚀、使用寿命长、降噪性能好、便于安装及维修的管材、管件，以及连接可靠、密封性能好的管道阀门设备。
7.3.4供暖系统宜采用适宜于干式工法施工的低温地板辐射供暖产品。
7.3.5当墙板或楼板上安装供暖与空调设备时，其连接处应采取加强措施。
7.3.6采用集成式卫生间或采用同层排水架空地板时，不宜采用低温地板辐射供暖系统。
7.3.7装配式混凝土建筑的暖通空调、防排烟设备及管线系统应协同设计，并应可靠连接。</td></tr><tr><td rowspan="2">4.4</td><td rowspan="2">电气和智能化</td><td>依据</td><td>《装配式混凝土建筑技术标准》GB/T 51231-2016第7.4.1~7.4.3条</td></tr><tr><td>要点/要求</td><td>7.4.1装配式混凝土建筑的电气和智能化设备与管线的设计，应满足预制构件工厂化生产、施工安装及使用维护的要求。
